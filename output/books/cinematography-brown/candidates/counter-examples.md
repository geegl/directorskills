title: 正面平光的扁平化陷阱
type: counter-example
source_chapter: 第7章 布光基础
source_quote: "正面平光是指主光离摄影机很近的情况，造成的结果是阴影很少甚至没有，深度很浅且不立体，拍出来的图像会非常扁平。"
summary: 正面平光（主光靠近摄影机轴线）是最常见的布光错误之一。它消除所有阴影，使主体看起来像贴在背景上的纸片。证件照就是典型例子。正确做法：将主光移到侧面或后方，让阴影创造立体感。灯光明信片式的"平"不是因为不够亮，而是因为方向不对。
tags: [布光错误, 正面平光, 扁平感, 立体感缺失]
```

title: 先拍备选镜头的连续性灾难
type: counter-example
source_chapter: 第2章 摄影技法
source_quote: "如果你想要先拍备选镜头再拍主镜头，很有可能会遇到连续性的问题。"
summary: 新手导演常犯的错误：先拍特写和过肩镜头，最后才拍主镜头。结果是主镜头中的演员位置、道具、灯光与备选镜头不匹配，导致无法剪辑。正确顺序：先拍主镜头建立连续性框架，所有后续镜头都必须与主镜头保持一致。主镜头不必是远景，但必须拍摄完整的场景。
tags: [拍摄顺序, 连续性错误, 主镜头, 新手错误]
```

id: e03
title: 频闪：气体放电光源的隐形杀手
type: counter-example
source_chapter: 第16章 技术问题
source_quote: "任何使用交流电的电弧灯泡都会在输出光线时随着电流的波形产生光量波动。虽然肉眼一般看不见频闪，但却会显示在胶片上。"
summary: HMI灯和荧光灯在交流电下会产生频闪，肉眼看不见但胶片/传感器会记录下来。频闪的原因：灯光输出频率与摄影机帧率不匹配。错误做法：使用非晶体控制的发电机、不检查快门角度。正确做法：确保电源频率稳定、使用晶体控制摄影机、查询安全快门角度/帧率组合表。
tags: [频闪, HMI灯, 技术陷阱, 帧率匹配]
```

id: e04
title: 灰卡拍摄的常见错误
type: counter-example
source_chapter: 第16章 技术问题
source_quote: "如果你在这个蓝色月光中放置一张灰卡拍摄的话，胶片调色员将不得不把灰卡调回它原本的中性灰色。这可能就需要去除所有的蓝色，你在布光中所做的所有努力都白费了。"
summary: 常见错误：在场景灯光（如蓝色月光）下拍摄灰卡。结果：胶转磁调色员会将灰卡调回中性灰，从而去除了你精心设计的所有色彩效果。正确做法：用纯色标准光源（3200K或5500K）单独照射灰卡，两盏灯45度角均匀照射，无滤片无减光器，然后用拍摄场景的相同曝光参数拍摄。
tags: [灰卡错误, 色彩参考, 胶转磁, 色彩校准]
```

type: counter-example
source_chapter: 第3章 视觉语言
source_quote: "不要截断人物的脚。大体来说，画框应该位于膝盖附近或拍进整条腿，如果在脚踝处截断人物会显得很诡异。同样道理，不要截断他们的手腕。"
summary: 构图中的截断禁忌：不要在脚踝处截断人物（显得诡异）、不要从鼻子处截断脸部、不要截掉下巴而保留头顶（反过来可以——截掉额头以上的特写是正常的）。原因：人脸和颈部传达的信息远多于额头和发型。正确做法：画框应位于膝盖附近或拍进整条腿。
tags: [构图禁忌, 截断, 人物构图, 画框]

type: counter-example
source_chapter: 第6章 电影连续性
source_quote: "也有'连续性过度'的说法。有时，一位老道的剧本监督可能会执着于某个小细节，即使这并不是很碍眼的问题，于是他们就会影响拍摄进度。"
summary: 连续性过度是与连续性不足相反的错误：场记执着于观众不会注意到的小细节，导致拍摄进度被严重影响。正确做法：在"完美连续性"和"拍摄效率"之间寻找平衡。判断标准：如果观众在正常观看中不会注意到这个不接戏的问题，那么它可能不值得花时间修正。
tags: [连续性过度, 场记, 效率, 平衡]

id: e07
title: CTB滤片的光量陷阱
type: counter-example
source_chapter: 第12章 色彩
source_quote: "CTB的通光率只有36%，而85滤镜有58%的通光率。这意味着当你在CTB上损失了将近一档半的光量时，在CTO上只损失了2/3档光量。"
summary: CTB（蓝色色温滤片）将钨丝灯转为日光色温时损失近1.5档光量，而CTO（橙色）只损失2/3档。这意味着用CTB转换灯光时需要更多灯光功率来补偿。替代方案：在窗户上加85滤片（只损失2/3档窗外光）、在灯光上加半蓝+窗户加半85、或直接在室内使用日光白平衡灯具（HMI/LED）。
tags: [CTB, 光量损失, 色温转换, 效率问题]
```

title: 单镜头法的剪辑困境
type: counter-example
source_chapter: 第2章 摄影技法
source_quote: "如果在剪辑时发现这个场景的节奏推进得太慢——你的选择就很有限了。如果你手上只有从头到尾拍摄了几遍的单镜头，那你在剪辑时就几乎没有其他选择了。"
summary: 单镜头法（整个场景一个镜头拍完）如果使用不当会导致剪辑困境：如果节奏不对，你没有任何备选镜头可用。正确做法：即使决定使用单镜头，也必须拍摄备选镜头和旁跳镜头以防万一。"还是稳妥点"——单镜头可以是艺术选择，但不能是唯一选择。
tags: [单镜头, 剪辑困境, 风险控制, 备选镜头]
```
