id: c04
title: "双人镜头中不同肤色的分别处理"
type: case
source_chapter: 第8章
source_quote: "整体调整会对肤色白皙的女演员有利，但会导致男演员肤色发紫。"
summary: 白人女演员和深肤色男演员在同一镜头中，一级校色无法同时让两人肤色都理想。解决方案：用HSL选色+形状遮罩隔离男演员脸部，将其Gamma往黄色/绿色方向调整消除紫色。目标不是让两人肤色相同，而是让差异不分散观众注意力。
tags: [肤色差异, 双人镜头, HSL选色, 二级调色]
```

type: case
source_chapter: 第3章
source_quote: "过量的红色通道对应曝光过度的脸部。"
summary: 红色通道被不可挽回裁切的严重过曝素材。步骤：(1)检查RGB分量示波器确认裁切通道；(2)做最佳一级校色；(3)用RGB Mixer将绿色通道数据混合到红色通道填补空缺；(4)用HSL亮度限定器隔离高光区域限制混合范围；(5)最终校色。这是"最后一搏"的修复方法。
tags: [过曝修复, 通道混合器, RGB Mixer, 红色通道, 紧急修复]
```
