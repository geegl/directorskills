```yaml
id: GL-01
title: 剧情/非剧情 (Diegetic / Non-Diegetic)
type: glossary
source_chapter: 第七章
source_quote: "剧情是指影片中的人物世界和故事世界。非剧情是不属于人物故事世界当中的元素"
summary: >
  剧情声音（diegetic）：存在于故事世界中的声音，人物可以听到，
  包括画内声音（声源可见）和画外声音（声源在故事中但画面外）。
  非剧情声音（non-diegetic）：故事人物听不到的声音，
  如画外音旁白、戏剧配乐。这是电影声音最基本的分类。
tags: [glossary, diegetic, non-diegetic, narrative, basic-concept]
```

```yaml
id: GL-02
title: 幻听/去幻听 (Acousmatic / De-acousmatic)
type: glossary
source_chapter: 第七章
source_quote: "米歇尔·琼创造了'幻听'术语来表示听不见原始声源的声音。原先看不见声源的声音其声源被展现出来叫做'去幻听'"
summary: >
  幻听（acousmatic）：听到声音但看不到声源。
  去幻听（de-acousmatizing）：揭示之前不可见的声源。
  经典案例：《绿野仙踪》中桃乐茜拉开帘子揭示烟雾中的小人。
  这种从不知到揭示的戏剧效果可以逐步去掉神秘性。
tags: [glossary, acousmatic, michel-chion, reveal, mystery]
```

```yaml
id: GL-03
title: 心理声学/听觉心理学 (Psychoacoustics)
type: glossary
source_chapter: 第三章/第四章
source_quote: "声音是如何产生、如何到达我们的耳朵的呢？利用物理学、解剖学、神经生理学的概念来研究声音"

  心理声学研究人类如何感知声音，包括物理振动如何转化为
  神经信号和心理体验。涉及耳朵结构（外耳/中耳/内耳）、
  频率感知、响度感知、掩蔽效应、定位能力等。
  这是声音设计的科学基础，决定了声音选择的技术边界。
tags: [glossary, psychoacoustics, perception, physics, ear-anatomy]
```

```yaml
id: GL-04
title: 声音透视/空间感 (Sound Perspective)
type: glossary
source_chapter: 第七章
source_quote: "声音空间也有合适的听音视角。可以是纯粹空间上的，如在大厅混响下远处包厢里听管弦乐"
summary: >
  声音透视指声音随距离、空间和方向变化的感知特征。
  包括：距离感（远近的音量/频率/混响变化）、
  方向感（左右/前后/上下的定位）、空间感（大/小/开放/封闭）。
  声音透视必须与画面空间一致，否则观众会感到不协调。
tags: [glossary, perspective, spatial, distance, directionality]
```

```yaml
id: GL-05
title: 拟音 (Foley)
type: glossary
source_chapter: 第一章/第二章
source_quote: "打牛肉做拳击声、踩玉米淀粉做雪地行走声。其经验是：仔细倾听效果好的声音，不要去看'真实的'声源"
summary: >
  拟音是在录音棚中同步画面制作音响效果声的艺术。
  名称来自先驱杰克·福利（Jack Foley）。
  原理：观众认为同步的声音就是画面发出的，因此真实声源
  不一定比替代声源效果好。拟音师用日常物品创造
  与画面同步的声音，是声音设计中最具创造性的领域之一。
tags: [glossary, foley, sound-effects, synchronization, art]
```

```yaml
id: GL-06
title: 环境声/声景 (Ambience / Soundscape)
type: glossary
source_chapter: 第一章/第八章
source_quote: "环境声的作用不是引起注意，而是进一步营造氛围、唤起感觉。环境声是对观众做出剧情发展的潜意识暗示"
summary: >
  环境声（ambience）：特定空间和时间的持续背景声音，
  如风声、交通声、室内嗡嗡声。声景（soundscape）：
  更宏观的声音环境概念，包含所有声音元素的有机组合。
  环境声的主要功能是营造真实感和潜意识情绪暗示。
tags: [glossary, ambience, soundscape, background, atmosphere]
```

```yaml
id: GL-07
title: 掩蔽效应 (Masking Effect)
type: glossary
source_chapter: 第三章
source_quote: "降低声音感知的现象是掩蔽效应。当两个声音从同一个方向传来时掩蔽效应更强"
summary: >
  掩蔽效应指一个声音降低对另一个声音的感知。
  分为频率掩蔽（相近频率相互遮盖）和时间掩蔽
  （响声遮盖前后较轻的声音，包括正向和反向掩蔽）。
  应用：可用于掩盖剪辑点不连续，但也需避免有用信息的意外损失。
tags: [glossary, masking, psychoacoustics, frequency, temporal]
```

```yaml
id: GL-08
title: 互引/同步效应 (Entrainment)
type: glossary
source_chapter: 第四章
source_quote: "互引是我们的身体和思想与外部节奏同步作用的一种特殊状态，从而导致一种特殊的身体和心理状态"
summary: >
  互引（entrainment）指人体生理节律与外部节奏同步的现象。
  心跳、呼吸、脑波会与音乐或环境节奏趋于一致。
  这解释了为什么节奏强劲的音乐能提升体温，
  慢节奏音乐能降低体温和放松状态。声音设计师可
  利用此效应引导观众进入特定生理和情绪状态。
tags: [glossary, entrainment, rhythm, synchronization, physiology]
```

```yaml
id: GL-09
title: 音头/音体/音尾 (Attack / Body / Decay — Envelope)
type: glossary
source_chapter: 第三章
source_quote: "音型参数即声音的包络，由音头（产生、发展）、音体（稳定状态、持续）、音尾（衰减、结束）组成"
summary: >
  声音包络描述声音的时间形态：音头（attack）是声音的起始上升段，
  音体（body/sustain）是持续段，音尾（decay/release）是衰减段。
  音头对识别声源至关重要——钢琴倒放后音头改变，
  听起来像手风琴。不同包络定义了不同乐器和声源的特征。
tags: [glossary, envelope, attack, decay, sustain, timbre]
```

```yaml
source_chapter: 第七章
source_quote: "和摄像机有景深一样，声音空间也有合适的听音视角。可以是纯粹空间上的，也可以是更主观的感觉"

  听音视角决定了观众与故事的情感距离和空间关系。
tags: [glossary, point-of-audition, subjectivity, perspective, spatial]
```

```yaml
id: GL-11
title: 预混/终混 (Pre-Mix / Final Mix)
type: glossary
source_chapter: 第一章
source_quote: "把声音元素分成音乐预混、音响预混、语言预混有三个主要原因：分组便于处理、声源方法不同、国际发行需要"
summary: >
  预混（pre-mix）：将大量独立声音分组处理为音乐/音响/语言三个
  预混磁底。终混（final mix）：将三个预混磁底合成为最终混录。
  预混决定了滤波、均衡、效果和声音位置的细节；
  终混决定各声音种类间的最终平衡。
tags: [glossary, pre-mix, final-mix, workflow, mixing]
```

```yaml
id: GL-12
title: 格式塔原理 (Gestalt Principle)
type: glossary
source_chapter: 第四章
source_quote: "格式塔心理学认为一种模式、一个统一体具有某些特殊的属性，不能够通过它的各种元素加在一起来得到"
summary: >
  格式塔原理描述大脑如何将零散感知组织为有意义的整体。
  听觉中的应用：轮廓/背景（从噪声中突出语言）、
  闭合（补全缺失声音）、邻近度（相近时间的声音分组）、
  相似度（相似音色的声音分组）、共同命运（同时变化的声音视为整体）。
tags: [glossary, gestalt, perception, psychology, organization]
```

```yaml
id: GL-13
title: 第一情绪/第二情绪 (First Emotion / Second Emotion)
type: glossary
source_chapter: 第八章
source_quote: "第一情绪是角色的情绪体验，第二情绪是观众的情绪反应。声音设计需同时考虑两者"
summary: >
  第一情绪：角色在故事中体验的情绪（人物的主观感受）。
  第二情绪：观众观看时产生的情绪反应（可能与角色不同）。
  声音设计需要在两者间找到平衡——既忠实于角色体验，
  又引导观众产生期望的情绪反应。两者可能一致也可能故意对立。
tags: [glossary, first-emotion, second-emotion, character, audience, empathy]
```

```yaml
id: GL-14
title: 音调中心 (Tonal Center)
type: glossary
source_chapter: 第五章
source_quote: "如果音乐停止了大概15秒以上，对于任何建立的音调中心的联结来说都是很不利的"
summary: >
  音调中心是音乐中给人"家"的感觉的核心音调。
  大脑在音乐停止后约15秒内仍保持对音调中心的记忆和参考，
  给影片带来统一效果。超过15秒则需要重新建立。
  声音设计师和作曲家应了解这个时间限制，
  或有意打破它以形成不和谐效果。
tags: [glossary, tonal-center, music, perception, 15-second-rule]
```

```yaml
title: 声音参考 (Sound Reference)
type: glossary
source_chapter: 第八章
source_quote: "声音参考可以是普遍的（如狮吼）、文化的（如鹅卵石路马蹄声）、历史的、或某一影片特有的（如《大白鲨》喀喀声）"

  文化的（特定社会/时期，如中东装饰音=异域）、
  历史的（时代特定，如拨号电话铃=过去）、
  电影特定的（某部片建立的，如大白鲨主题=危险）。
tags: [glossary, reference, universal, cultural, historical, film-specific]
```
