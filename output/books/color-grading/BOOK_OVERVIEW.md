# BOOK_OVERVIEW.md — 《调色师手册》第2版

## Adler 四步阅读法

### 第一步：这本书到底在谈什么？
本书是影视数字调色领域的专业参考书，系统覆盖了从一级校色（对比度/色彩平衡）、二级校色（HSL选色/形状遮罩）到镜头匹配、场景平衡、色彩风格创造的完整调色工作流。核心论点是：调色既是技术也是艺术——技术层面需要掌握示波器分析、信号合法性、色彩科学；艺术层面需要理解色彩记忆、观众心理、光影叙事。作者 Alexis Van Hurkman 以软件无关的通用方法论贯穿全书，辅以大量真实影片案例截图。

### 第二步：这本书具体说了什么？（按章节）
1. **调色师的角色**（第1章）：调色师是视觉叙事的最后一环，职责包括精炼情绪、创造风格、打磨镜头。
2. **调色环境设定**（第2章）：监视器选择（BT.709/DCI P3/Rec.2020色域）、色温标准（D65/D93）、伽马设置、调色间布置（18%灰环绕墙）、投影机配置。
3. **一级校色：对比度**（第3章）：Lift/Gamma/Gain三大控件、Offset/Exposure、亮度曲线、Contrast/Pivot工具、Log调色模式控件（Shadow/Midtone/Highlight）、高动态范围素材处理、过度曝光素材修复（通道混合器重建）。
4. **一级校色：色彩**（第4章）：色温原理（开尔文）、色彩平衡控件的重叠范围、通道偏移（Offset/Printer Points）、RGB曲线工具、饱和度调整、色彩对比概念。
5. **HSL选色与色相曲线**（第5章）：HSL限定器的创建与优化、高质量素材vs高压缩素材的键控差异、组合蒙版、特定影调范围饱和度调整。
6. **形状遮罩**（第6章）：椭圆/矩形/渐变遮罩、B样条与贝塞尔曲线绘制、羽化与柔化、暗角技术、运动跟踪与roto。
7. **关键帧动画调色**（第7章）：色彩动画、光线变化模拟、创意性调色动画（如《单身男子》风格）。
8. **色彩记忆**（第8章）：观众对肤色/天空/植物的记忆色彩偏好、肤色的生理学基础（黑色素/血红蛋白/β-胡萝卜素）、I-bar肤色指标、天空色彩的物理原理（瑞利散射）、日落/晨光的色彩特征。
9. **镜头匹配与场景平衡**（第9章）：评估差异→对比度→色彩平衡→饱和度→异常检查→迭代调整的工作流。
10. **质量控制与广播安全**（第10章）：亮度/色度合法范围、RGB分量示波器检查、饱和度合法化、图形与动画的安全色彩。
11. **风格宝典节选**（第11章）：着色与偏色、暗部色调、自然饱和度（Vibrance）、目标饱和度。

### 第三步：这本书说得有道理吗？
**有道理的部分：**
- 示波器优先的工作方法论（波形/矢量/分量示波器）是行业共识
- "先调整对比度再调整色彩"的操作顺序被广泛验证
- 肤色记忆色彩研究（Bartleson等）有扎实的学术支撑
- Log编码素材的调色策略（先在Log状态做整体调整，再正常化做细节）是专业实践

**值得商榷的部分：**
- 书中部分示波器数值以NTSC/IRE为基准，不同地区标准有差异
- 对某些软件功能的描述（如DaVinci Resolve）可能随版本更新而变化
- 色彩偏好的研究样本多来自西方文化，跨文化适用性需谨慎

### 第四步：这本书跟我有什么关系？
对AI影视工作流的价值：
- **多镜头一致性**：镜头匹配方法论可直接指导AI生成视频的色彩一致性prompt策略
- **电影感Look**：着色/偏色/暗部色调等技术可转化为AI视频后期的风格化prompt
- **色彩科学基础**：色温、饱和度、对比度的精确描述词汇可提升AI生成的精准度
- **质量控制思维**：广播安全概念可映射到AI视频输出的质量约束

## 关键术语速查
| 术语 | 定义 |
|------|------|
| Lift/Gamma/Gain | 暗部/中间调/高光的对比度三控件 |
| Offset | 整体偏移，同时移动所有色彩通道 |
| HSL Qualifier | 基于色相/饱和度/亮度的选色工具 |
| Vectorscope | 矢量示波器，显示色相和饱和度 |
| Waveform Monitor | 波形监视器，显示亮度分布 |
| I-bar | 矢量示波器上的肤色参考线 |
| Log编码 | 保留高动态范围的对数编码方式 |
| LUT | 查找表，用于色彩空间转换或风格应用 |
| 广播安全 | 视频信号在合法亮度/色度范围内 |
| Power Window | 达芬奇Resolve中的形状遮罩工具 |

## 结构评估
- **章节逻辑**：从环境→基础工具→高级技术→艺术应用→质量控制，由浅入深
- **实操占比**：约60%为具体操作步骤，30%为理论原理，10%为行业背景
- **跨域价值**：色彩记忆章节（第8章）连接了视觉心理学与调色实践，是跨域价值最高的部分
