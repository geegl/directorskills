```yaml
  应从"技师"发展为"技术型艺术家"，最终成为"叙事者"。
```

```yaml
id: PR-02
title: 第一遍阅读心耳原则 (First Read — Inner Ear Principle)
type: principle
source_chapter: 第一章
source_quote: "你应该从完成的剧本中第一次'听'到作品的声音。不要让你那种想看画面的冲动破坏这个独一无二的机会"
summary: >
  声音设计必须从第一遍剧本阅读开始，不看画面，用"心耳"想象声音。
  一旦看过画面，就无法回到未受视觉影响的初始状态。第一遍阅读要
  一口气完成，保持与影片节奏一致（约一分钟一页），标记关键词和
  声音想法，但不要停下来分析。
tags: [principle, script-reading, inner-ear, first-impression, workflow]
```

```yaml
id: PR-03
title: 少即是多原则 (Less Is More Principle)
type: principle
source_chapter: 第四章
source_quote: "有时候你可以用最少数量的声音元素得到最大的冲击力，通过让观众填补自己头脑中的信息缺失"
summary: >
  用最少的声音元素获得最大冲击力。让观众自行填补信息空白，
  使他们有更大参与感。本·伯特设计黑武士声音时，经过大量组合实验后，
  发现单一的呼吸循环声最具冲击力。过度堆砌声音反而降低效果。
tags: [principle, less-is-more, simplicity, audience-participation, ben-burtt]
```

```yaml
id: PR-04
title: 对比产生戏剧效果 (Contrast Creates Drama)
type: principle
source_chapter: 第一章/第三章
source_quote: "关键不在于音量，而在于变化和对比。避免持续的大音量声音，因为这会导致观众没有听觉放松"
summary: >
  戏剧效果来自对比，而非绝对值。响与轻、快与慢、满与空的对比
  创造张力。持续大音量导致听觉疲劳（听觉反应降低灵敏度）。
  乔治·沃特在《珍珠港》中在爆炸段落间留出喘息空间。
  声音设计师必须管理观众的听觉"呼吸"。
tags: [principle, contrast, dynamics, loudness, fatigue, pearl-harbor]
```

```yaml
id: PR-05
title: 环境声潜意识暗示原则 (Ambience as Subliminal Cue)
type: principle
source_chapter: 第一章
source_quote: "在一部好的电影里，环境声是对观众作出剧情发展的潜意识暗示；环境声的作用不是引起注意，而是进一步营造氛围"
summary: >
  环境声的功能是潜意识层面的叙事暗示，而非引起注意。
  环境声的音量和成分变化应跟随剧情曲线（dramatic arc），
  在不同时间段录制不同热闹程度的环境声，在混录时调整
  以配合戏剧性转场。好的环境声"隐身"却有效。
tags: [principle, ambience, subliminal, atmosphere, dramatic-arc]
```

```yaml
id: PR-06
title: 声画同步幻觉原则 (Sound-Image Sync Illusion)
type: principle
source_chapter: 第一章
source_quote: "观众会认为任何与画面同步的声音，就是该画面发出的声音。一些真实的声音戏剧效果并不好，而完全不同声源的声音效果却不错"
summary: >
  两个关键现象：1）观众认为同步的声音就是画面发出的；
  2）真实声音效果往往不好，替代声源反而更有效。
  这是拟音艺术的基础——打牛肉做拳击声、踩玉米淀粉做雪地声。
  声音设计师应倾听效果好的声音，而非追求"真实"声源。
tags: [principle, synchronization, foley, illusion, perception]
```

```yaml
id: PR-07
title: 声音发展线原则 (Sound Developmental Line)
type: principle
source_chapter: 第一章/第八章
source_quote: "当一个声音元素被重复并和叙事主题相关联时，在人物的转化阶段，声音会有一个推动戏剧发展的涵义的累积"
summary: >
  声音必须随叙事发展而变化，不能从头到尾保持不变。
  重复的声音元素与叙事主题关联后，会在人物转化时积累含义。
  丹尼·戴维斯在《黑客帝国》中确保每场打斗的声音都不同，
  体现了声音的戏剧发展线。
tags: [principle, development, evolution, repetition, dramatic-arc, matrix]
```

```yaml
id: PR-08
title: 频率分离原则 (Frequency Separation Principle)
type: principle
source_chapter: 第一章/第五章
source_quote: "音乐在频率上可以对立，使得声音元素能很容易地在感知上分开；也可以在音调上非常接近，使乐器声音能从效果声当中出现"
summary: >
  音乐与音效之间需要频率上的协调——可以对立分离，也可以故意接近融合。
  丹尼·戴维斯在《黑客帝国》中做了"跳棋盘"，使焦点在音乐和音效间
  转换但不完全分开。大爆炸时音乐避开低频，玻璃碎裂时避开高频。
tags: [principle, frequency, separation, mixing, music-sfx, matrix]
```

```yaml
id: PR-09
title: 情绪冲击原型原则 (Emotional Impact Archetype)
type: principle
source_chapter: 第二章
source_quote: "当我们发现或创造听起来带有'情绪外壳'、具有歌声性质的非人类声音时，这种原型声音模版会得到我们的喜欢"
summary: >
  具有人类声音特征（空气摩擦、呼吸感）的非人类声音会产生情绪共鸣。
  尖叫声可从压缩空气管得到，叹气声可从老木门得到。
  在动物声上叠加人声可增加情感层：狮吼=力量，黑猩猩=古怪，
  猫叫=诱惑。人声需保持在不明显的电平。
tags: [principle, archetype, emotion, voice, sound-design, creation]
```

```yaml
id: PR-10
title: 声音一致性原则 (Sound Consistency Principle)
type: principle
source_chapter: 第八章
source_quote: "一旦某个声音被确定成为那个声画对象或环境声的参考，就应该坚持使用这个参考，除非故事里发生了明确的变化"
summary: >
  一旦建立了声画关联，就必须保持一致。耳朵渴望关联，
  随意改变会让人觉得"不对劲"。从音效库随便找个声音
  可能颠覆观众已形成的精确声画关联。《筹码》中用错误的
  摩托车声配哈雷引起了观众愤怒。
tags: [principle, consistency, reference, sound-image, association]
```

```yaml
id: PR-11
title: 听音视角原则 (Point of Audition Principle)
type: principle
source_chapter: 第七章
source_quote: "和摄像机有景深一样，声音空间也有合适的听音视角。可以是纯粹空间上的，也可以是更主观的感觉"
summary: >
  像摄影机有景深一样，声音也有"听音视角"。可以是空间性的
  （如远处包厢听管弦乐 vs. 指挥台旁），也可以是主观的
  （如人物听到的电话声经过滤波处理）。这决定了观众的
  空间感和情感距离。
tags: [principle, point-of-audition, subjectivity, spatial, perspective]
```

```yaml
id: PR-12
title: 声音大小控制原则 (Sound Size Control)
type: principle
source_chapter: 第七章
source_quote: "声源的外观大小可以通过改变原始素材的速度来控制。在《终结者2》中，沸腾液体声音速度被降低，让人觉得熔化槽更大"
summary: >
  降低声音速度可使声源显得更大，提高速度可使声源显得更小。
  《终结者2》中将液体声降速表现巨大熔化槽，同时混入原始速度
  的声音保持与画面动作同步。混录中音量、扬声器位置的选择
  也影响空间大小感知。
tags: [principle, size, speed-change, perspective, terminator2]
```

```yaml
id: PR-13
title: 预期违背原则 (Expectation Violation Principle)
type: principle
source_chapter: 第五章
source_quote: "当期望不准确的时候，情绪会在音乐和故事当中爆发。通过某一时刻与预期的背离，下一个应验的预期时刻能够被强化"
summary: >
  音乐和叙事的力量来自预期的建立和违背。预期被满足产生愉悦，
  被违背产生情绪爆发（惊喜/恐惧/幽默）。违背太大导致不连贯，
  每次都满足则导致厌烦。恐怖片中先用虚惊建立预期，
  再用真正吓人的声音打破预期，效果加倍。
tags: [principle, expectation, surprise, tension, release, horror]
```

```yaml
id: PR-14
title: 互引效应原则 (Entrainment Principle)
type: principle
source_chapter: 第四章/第五章
source_quote: "互引是我们的身体和思想与外部节奏同步作用的一种特殊状态。通过互引效应，心跳的节奏会跟音乐的节奏趋于一致"
summary: >
  互引效应（entrainment）指人体生理节律（心跳、呼吸、脑波）
  与外部声音节奏同步的现象。慢节奏（64BPM/接近安静心跳）
  促进放松和深度思考；快节奏引起兴奋和急促呼吸。
  萨满鼓声可引导脑波进入Theta状态。声音设计师可利用此效应
  引导观众生理和情绪状态。
tags: [principle, entrainment, rhythm, heartbeat, brainwave, physiological]
```
