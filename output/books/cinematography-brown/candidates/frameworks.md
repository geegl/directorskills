```yaml
title: 三点布光系统
type: framework
source_chapter: 第7章 布光基础
source_quote: "主光：打在人物或物体上的关键光线。补充光：照亮主光照不到的阴影部分。背光：从人物或物体背后射来的光，会在主体边缘形成一圈亮边。"
summary: 三点布光是电影摄影最基础的灯光拓扑：主光（Key Light）定义主要照明方向和阴影；补光（Fill Light）控制阴影深度和对比度；背光（Rim/Back Light）将主体从背景中分离出来。三者的强度比例（灯光比）决定了场景的情绪基调。高补光比=低调/戏剧性，低补光比=高调/轻快。
tags: [布光, 三点照明, 灯光比, 基础框架]
```

```yaml
id: f03
title: 硬光/软光二元框架
type: framework
source_chapter: 第7章 布光基础
source_quote: "归根结底布光中其实只有两种光线（就光线的品质而言）：硬光和软光。光线的硬和软取决于光源的大小。"
summary: 灯光品质的唯一决定因素是光源的相对大小：光源越小→光线越硬→阴影越清晰；光源越大→光线越软→阴影越模糊。太阳虽大但远所以是硬光源；阴天整片天空是软光源。片场可通过反射（白泡沫板）或透射（柔光布）将硬光变软光。这是所有布光决策的第一步判断。
tags: [灯光品质, 硬光, 软光, 光源大小]
```

```yaml
title: 主场景拍摄法（Master Scene Method）
type: framework
source_chapter: 第2章 摄影技法
source_quote: "主场景法非常简单：首先，你从头至尾将整个场景拍摄下来——这就是主镜头。当你拍好主镜头后，就开始拍预备选镜头。"
summary: 主场景法是最常用的场景拍摄策略：1）先拍完整主镜头建立连续性框架；2）再拍备选镜头（过肩、中景、特写）；3）先完成一侧所有镜头再换到另一侧；4）反应镜头保持与主镜头相同的焦距和尺寸。核心原则：主镜头先拍，因为所有后续镜头必须与主镜头保持一致。
tags: [拍摄技法, 主场景法, 连续性, 覆盖策略]
```

```yaml
id: f06
title: 灯光方向与情绪映射
type: framework
source_chapter: 第7章 布光基础
source_quote: "如果大部分的光线是从侧面或后面照来，画面就会看上去更'黑'，更有神秘感和戏剧感。避免正面平光——正面平光是指主光离摄影机很近的情况。"
summary: 灯光方向直接映射情绪效果：前光/正面光→扁平、无立体感（证件照感）；3/4前光→自然、中性；侧光→戏剧性、明暗对比强烈；3/4背光→神秘、深度感强；背光→轮廓感、分离感。关键原则：主光越从侧面或背面打来，主体深度和立体感越强。从后场打光总比前场打光好。
tags: [灯光方向, 情绪映射, 布光原则, 后场打光]
```

```yaml
id: f07
title: 叠光技术（Motivated Lighting）
type: framework
source_chapter: 第7章 布光基础
source_quote: "台灯照射的地方再设置一盏小灯，让它以和台灯一样的角度照射演员。增添的那盏光源需要和台灯拥有一样的品质（硬或软）和一样的色调。"
summary: 叠光是让人工灯光看起来像场景内真实光源发出的光的技术。核心方法：在真实光源（台灯/窗户/壁炉）照射方向上设置额外灯具，匹配真实光源的品质（硬/软）、方向和色温。解决的问题：真实光源太暗无法正确曝光演员，或真实光源过曝导致灯泡细节丢失。关键：叠光必须与真实光重叠。
tags: [叠光, 光源光, 布光技巧, 真实感]
```

```yaml
id: f08
title: 色温转换滤片系统
type: framework
source_chapter: 第12章 色彩
source_quote: "色温的单位是开尔文度，简称大K。色温超过5000K是冷色（略带蓝色的白色），而更低的色温（2700—3000K）叫做暖色。"
summary: 色温系统提供精确的色彩控制框架：日光≈5500K，钨丝灯≈3200K。CTO（橙色）将冷光变暖，CTB（蓝色）将暖光变冷。迈尔德参数（1,000,000÷K）提供精确计算：3200K=312迈尔德，5500K=182迈尔德，差值=130→用85/CTO滤片。此系统让灯光色温匹配从猜测变为精确计算。
tags: [色温, 滤片, CTO, CTB, 迈尔德]
```
