```yaml
id: p01
title: 灯光比原则
type: principle
source_chapter: 第7章 布光基础
source_quote: "场景的灯光有时会用主/补光比作为评价标准。低调光：阴暗且黯淡的光线，补光很少甚至没有，也可以说是主/补光比很高的光。"
summary: 灯光比（主光与补光的强度比）是控制场景情绪的核心参数。高灯光比（如4:1或8:1）=低调光，产生戏剧性、神秘感、黑色电影风格；低灯光比（如2:1或1:1）=高调光，产生明亮、轻快、时尚感。灯光比可通过测光表分别测量亮面和暗面来精确控制。
tags: [灯光比, 高低调光, 情绪控制, 对比度]
```

```yaml
id: p02
title: 动机照明原则（Motivated Lighting）
type: principle
source_chapter: 第7章 布光基础
source_quote: "我们通常希望达到台灯照亮了主体的效果，但因为某些原因，它就是做不到。如果我们将台灯亮度调高照亮演员，那阴影部分会彻底过黑。"
summary: 动机照明的核心原则是：所有人工灯光都应看起来有合理的光源出处。观众需要理解光从哪里来——窗户、台灯、壁炉、蜡烛。当真实光源不足以正确曝光时，使用叠光技术：在画面外设置额外灯具，与真实光源保持相同方向、品质和色温。这避免了"光从天而降"的不自然感。
tags: [动机照明, 真实光源, 自然感, 叙事逻辑]
```

```yaml
id: p03
title: 色温匹配原则
type: principle
source_chapter: 第12章 色彩
source_quote: "改变色彩平衡是最简单最可行的图像控制方法。摄影机有内置的色彩平衡，可以设置成日光白平衡或钨丝灯平衡。"
summary: 色温匹配是灯光工作的基础原则：同一场景中所有光源的色温必须匹配或有意识地不匹配。匹配=专业感；不匹配=必须有叙事理由（如荧光灯的偏绿暗示不安）。日光5500K与钨丝灯3200K是两个标准参考点。荧光灯因不连续光谱需要特殊处理（减绿片/CC滤片）。
tags: [色温匹配, 白平衡, 荧光灯矫正, 色彩一致性]
```

```yaml
id: p04
title: 从后场打光原则
type: principle
source_chapter: 第7章 布光基础
source_quote: "当我们为一个对话场景布光时，从后场向演员打光总是一个好办法。后场指的是远离摄影机拍摄的那一边。"
summary: 从后场（远离摄影机的一侧）打光优于从前场（摄影机同侧）打光。后场打光产生好看的面部阴影、烘托脸部轮廓、增加立体感。前场打光产生扁平、无深度的"证件照"效果。这是几乎所有对话场景布光的首选方向。即使不能完全从后场打光，也应尽量让主光偏向后场。
tags: [后场打光, 布光方向, 立体感, 对话场景]
```

```yaml
id: p05
title: 180度法则（分割线原则）
type: principle
source_chapter: 第6章 电影连续性
source_quote: "观众的位置（或者说摄影机的位置）划下了这条线，有时叫作180度线或运动轴线。"
summary: 180度法则是连续性的核心原则：两个角色之间的连线定义了一条轴线，摄影机必须始终在轴线的同一侧拍摄。违反此法则会导致画面方向反转，观众会困惑角色位置关系。例外情况：中轴镜头（物体正对摄影机运动）可用于过渡到另一侧；在空间布局清晰的场景（如法庭）中可以灵活处理。
tags: [180度法则, 连续性, 轴线, 画面方向]
```

```yaml
id: p06
title: 平方反比定律
type: principle
source_chapter: 第10章 曝光
source_quote: "光量的减少与其与光源距离的平方成正比。通俗来说，当你与光源的距离增加一倍时，获得的光量就是原来的1/4。"
summary: 平方反比定律是灯光强度计算的基础物理原理：距离加倍→光量变为1/4（损失2档曝光）。灯距物体11英尺移到8英尺可增加一档曝光。严格来说仅适用于点光源，但聚光灯在较远距离下也适用。此定律是片场快速估算灯光调整量的核心工具。
tags: [物理定律, 光量计算, 灯光距离, 曝光估算]
```

```yaml
id: p07
title: 画面方向连续性原则
type: principle
source_chapter: 第6章 电影连续性
source_quote: "运动的无缝连接。不管是打开一扇门、拿起一本书或是停好一辆车，前后两个镜头中的运动必须没有间隙。"
summary: 画面方向连续性有四层：内容连续性（道具/服装一致）、动作连续性（运动无缝衔接）、位置连续性（道具位置不变）、时间连续性（时间流逻辑一致）。核心原则：不能让观众产生混淆。所有演员动作都必须重叠拍摄，在镜头开头和结尾为剪辑提供额外重叠动作。
tags: [连续性, 画面方向, 剪辑, 重叠拍摄]
```

```yaml
id: p08
title: 视觉比重与构图平衡原则
type: principle
source_chapter: 第3章 视觉语言
source_quote: "构图中的每个视觉元素都有其视觉比重，他们能够组成平衡或不平衡的构图。一个物体的视觉比重主要是由它的大小决定的，但也受到其他因素的影响。"
summary: 视觉比重由大小、位置、颜色、运动和属性决定。平衡构图=正式、庄重感；不平衡构图=紧张、不稳定感。三分法则是最简单的平衡工具：将画面三等分，四个交点是观众注意力的优先点。鼻前空间（视线空间）和头上空间是人物构图的关键参数。
tags: [构图, 视觉比重, 平衡, 三分法则]
```

```yaml
id: p09
title: 硬光来自小光源原则
type: principle
source_chapter: 第7章 布光基础
source_quote: "光源越小，光线就越硬，这是至关重要的一点。光线的硬和软取决于光源的大小。"
summary: 光线品质的唯一决定因素是光源的相对大小（非绝对大小）。太阳虽巨大但距离远→天空中显得很小→硬光。阴天整片天空→巨大光源→软光。片场制造软光的两种方法：1）在大面积白色物体上反射（泡沫板越大光线越软）；2）让光通过柔光布（布越厚光线越软）。
tags: [光源大小, 硬光软光, 物理原理, 柔光技术]
```

```yaml
id: p10
title: 避免正面平光原则
type: principle
source_chapter: 第7章 布光基础
source_quote: "正面平光是指主光离摄影机很近的情况，造成的结果是阴影很少甚至没有，深度很浅且不立体，拍出来的图像会非常扁平。"
summary: 正面平光（主光靠近摄影机轴线）是应避免的布光方式，因为它：1）消除阴影→失去立体感；2）无法将主体从背景中分离；3）看起来像证件照。例外：高调时尚/美容摄影可能故意使用。基本原则：主光越从侧面或背面打来，主体的深度和立体感越强。
tags: [正面平光, 布光禁忌, 立体感, 分离感]
```

```yaml
id: p11
title: 动机照明（光源光）原则
type: principle
source_chapter: 第7章 布光基础
source_quote: "光源光：当场景中有确定光源的光，例如窗户、台灯、壁炉等。在某些情况下，这些光的光源就在场景中。"
summary: 光源光（Practical Light）原则要求场景中的光线必须有可见或可推断的光源出处。台灯、窗户、壁炉、蜡烛等都是合理的光源。当真实光源不足以正确曝光时，使用叠光技术补充，但必须保持方向、品质和色温一致。每个真实光都应装减光器进行微调。
tags: [光源光, 动机照明, 叠光, 真实光源]
```

```yaml
id: p12
title: 薄暮现象与月光色温偏见
type: principle
source_chapter: 第12章 色彩
source_quote: "月光只是经过反射了的太阳光，和白天的阳光的颜色是一样的。这就是为什么在电影中，传统上会用略带蓝色的灯光给夜景布光，这体现了我们对月光的感知，而不是在自然界中的真实情况。"
summary: 薄暮现象（Purkinje Effect）：人眼在昏暗光线下对蓝色更敏感、对红色不敏感，导致我们感知月光偏蓝（实际月光与日光同色温）。电影摄影利用这一心理偏见：夜景用偏蓝灯光布光，不是因为月光真是蓝色的，而是因为观众"感觉"月光是蓝色的。这是用科学解释"为什么可以打破物理真实"的典范。
tags: [薄暮现象, 月光, 色温偏见, 视觉心理学]
```
