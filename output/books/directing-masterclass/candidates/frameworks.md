# 候选框架 — 思维模型 / 决策框架 / 推理方法

> 提取自《电影导演大师课》全书，宁多勿漏，待阶段 1.5 验证筛选。

- id: f01
  title: "很久很久以前"故事构建框架
  type: framework
  source_chapter: 第 9 章
  source_quote: "很久很久以前……有一位……他/她……但是……有一天，事情发生了……因此，接下来……但与此同时……为……所不知的是……直到……的时候……突然，出乎意料的是……结果是……从这以后……"
  summary: 一套用常用语引导的逐步构建叙事的框架。从设定时间地点和主角开始，引出障碍和目标，再通过攻击点启动叙事推进，经历并行发展和戏剧反讽，最终通过突转走向冲突解决和结局。每个步骤都对应戏剧结构的一个核心要素，可反复应用于任何故事类型。
  tags: [故事结构, 叙事构建, 编剧, 通用]

- id: f02
  title: 活动 vs. 动作区分判断
  type: framework
  source_chapter: 第 10 章
  source_quote: "一项'活动'是现在时态，即某个人正处于做某件事的过程中。它指的是正在进行的状态，不包含开头、中间和结尾。而'动作'，则包含内在的戏剧性，包括整个完成了的活动，意味着正在做的这件事可能产生一种效果。"
  summary: 一个二分判断框架，用于区分银幕上"正在进行的活动"（未完成时态，无戏剧性）和"完整的动作"（完成时态，包含开端-发展-结果，推动故事前进）。面对任何场景元素时自问：这是活动还是动作？电影必须依赖动作来制造张力，活动缺乏戏剧力量。
  tags: [场景分析, 剧本诊断, 活动-动作, 决策树]

- id: f03
  title: 获得创意的四步法
  type: framework
  source_chapter: 第 5 章
  source_quote: "1.搜集资料……2.整理资料……3.孵化素材……4.呵护思维火花。搜集资料的活动不应与其后的步骤相混淆。接下来的这个步骤则重在分析和评价。"
  summary: 创意产生的四阶段流程：（1）搜集资料——大量调研、开放头脑；（2）整理资料——分析归类、辨识模式；（3）孵化素材——等待潜意识加工（"睡在上面"）；（4）呵护火花——在激情和技巧之间保持平衡，通过大量练习让技巧成为条件反射，解放想象力。缺少任何一步都会导致创作失败。
  tags: [创意方法, 写作流程, 思维模型, 创作过程]

- id: f04
  title: 戏剧反讽操控法
  type: framework
  source_chapter: 第 11 章
  source_quote: "戏剧反讽则被定义为'通过引导观众了解实际情境与相应台词之间的反差，并且让剧中角色对这种反差一无所知，从而达到一种戏剧效果'。"
  summary: 一个叙事张力操控框架。导演通过让观众知道角色不知道的关键信息（"桌子底下的炸弹"），在对话和动作中植入悬而未决的张力。观众被置于焦急等待爆发的状态中，普通的场景也变得刺激紧张。希区柯克的炸弹比喻是核心模型：悬念 > 惊吓。运用时机：故事早期安放信息，中段维持等待的焦虑，高潮引爆。
  tags: [叙事张力, 悬念技巧, 导演技法, 观众心理]

- id: f05
  title: 隐身见证人视点操控
  type: framework
  source_chapter: 第 18 章
  source_quote: "我创造了一个角色——想象中无所不在的隐身见证人，它能够代表从事这个阶段工作的导演。我用这个角色来象征导演的心灵之眼与耳，它在摄影机创造的想象时空中四处出没。"
  summary: 一套导演决策的思维工具。在拍摄现场，导演应想象一个可在虚构时空中自由移动的"隐身见证人"，它代表观众的好奇心。每一个机位、景别和剪切决策都应回答：此刻见证人想站在哪里看什么？这个框架将复杂的摄影机决策简化为一个统一的视点选择问题。
  tags: [导演思维, 视点选择, 摄影决策, 想象工具]

- id: f06
  title: 电影大纲分析法（逆向骨架提取）
  type: framework
  source_chapter: 第 7 章
  source_quote: "电影大纲是一种工作文件……是对一部已经存在的电影长片的情节结构的简析，是故事各个步骤的简要大纲，是一种工具，能够用来拆解并揭示戏剧叙事结构及其中起作用的机制。"
  summary: 一种逆向工程分析工具。将任何已完成的电影剥去血肉（画面、对白、表演），只保留骨架（事件链和因果逻辑）。用编号段落记录每个场景，每段以"因此，这一事件的结果是……"开头，强迫发现因果链条。可诊断结构缺陷、发现多余场景、评估叙事节奏。
  tags: [结构分析, 剧本诊断, 逆向工程, 学习工具]

- id: f07
  title: 期待与不确定的张力公式
  type: framework
  source_chapter: 第 2 章/第 9 章
  source_quote: "戏剧几乎总是带来'期待混杂着不确定'的效果。'期待'指的是对于刚刚发生的事件将如何促发我们预料随后可能发生的新事件，我们所怀有的期待感，'不确定'是指尽管我们知道接下来会发生什么，但却不能肯定它的结果会怎样。"
  summary: 戏剧张力的核心公式：张力 = 期待（anticipation）x 不确定（uncertainty）。两个因素缺一不可——纯粹的期待会变成无聊的等待，纯粹的不确定会变成困惑。导演的任务是持续操控这两个变量，在给出答案的同时制造新问题，形成"中国套盒"式的嵌套悬念结构。
  tags: [戏剧理论, 张力模型, 叙事动力, 核心公式]

- id: f08
  title: 剪辑时机判断——预备-动作-反应模型
  type: framework
  source_chapter: 第 10 章/第 25 章
  source_quote: "你可能要多花些工夫去强调和发挥事件的预备阶段和遗留效应，至少不亚于处理事件本身所花的时间。有经验的漫画家对动作发生的那一刻的兴趣远不如对事件前后发生的兴趣。"
  summary: 一个剪辑决策框架。对于任何动作场景，将其分为三个阶段：预备（能量积聚）、动作本身（瞬间爆发）、反应（遗留效应）。剪辑师应把最多精力放在预备和反应上，动作本身常常被快速带过甚至省略。这个框架直接指导剪切时机的选择：在动作完成前切出（悬念），或在动作完成后切入反应（震惊）。
  tags: [剪辑决策, 时机判断, 动画原理, 场景节奏]

- id: f09
  title: 角色反转练习（反派视角重述）
  type: framework
  source_chapter: 第 7 章
  source_quote: "让作者临时进行一个角色反转的游戏，以反面人物的视角即兴创作出故事的新版本，这个反面角色作者很难确定，但一旦确定，就能被证实是缺失的那个关键元素。"
  summary: 一种诊断和创作工具。当故事缺乏叙事动力时，强迫自己从反面角色的视角重新讲述故事。这个练习能揭示：（1）反派是否真正有驱动力；（2）主角与反派之间的冲突是否足够强烈；（3）故事缺失的关键元素。源自心理剧治疗方法中的角色互换技术。
  tags: [角色设计, 叙事诊断, 视角转换, 创作练习]

- id: f10
  title: 潜台词揭示——声画对立法
  type: framework
  source_chapter: 第 6 章/第 14 章
  source_quote: "画面与对白的表面意思产生矛盾对立的部分，是影片最有冲击力的部分。对白所表达的内涵远甚于其字面意思。"
  summary: 一种通过电影语法揭示潜台词的方法论。导演在剪辑台上可以通过选择反应镜头的时机、调整画面与对白的关系，让银幕上角色说的内容与其内心想法产生矛盾。对白的字面意思说的是A，但镜头呈现给观众的信息暗示的是B——这种声画对立就是电影潜台词的核心载体。
  tags: [潜台词, 声画对立, 导演技法, 表演层次]

- id: f11
  title: 多米诺因果推进法
  type: framework
  source_chapter: 第 2 章/第 7 章
  source_quote: "大多数的强情节故事都是基于因果关系来构筑的。每个事件都像一块多米诺骨牌，向前倾倒，撞倒下一块，把观众牢牢地置于期待之中。'然后呢？'每个场景都展现一场小危机，带来新的不确定性。"
  summary: 叙事推进的因果链模型。每个场景/事件必须是前一个事件的"因此"（结果），同时是下一个事件的"导火索"。用"因此，这一事件的结果是……"作为每个场景开头，强迫检查因果链条的完整性。如果某场戏无法以这种方式连接，它可能不属于这个故事。
  tags: [因果逻辑, 叙事动力, 场景诊断, 结构完整性]

- id: f12
  title: 清晰性优先原则
  type: framework
  source_chapter: 第 4 章
  source_quote: "'清晰'就是传递关键信息，过滤非关键信息。要有极大的才智和洞见才能将重要的内容从混乱而过分复杂的材料中分离出来，加以保留并进行强调。"
  summary: 一个编剧决策框架。面对任何场景或对白时做两层判断：（1）这是关键信息还是非关键信息？（2）非关键信息是否被恰当地淡化处理？清晰不等于直白——"模棱两可"只有在多重含义都清晰表达时才有价值。含糊不是深刻，精确才是力量。
  tags: [编剧原则, 信息管理, 清晰性, 编辑决策]

- id: f13
  title: 必备场景（Scene a Faire）识别法
  type: framework
  source_chapter: 第 12 章
  source_quote: "必备场景是观众差不多是明确地、有意识地预见或想要的一个场景，它的缺失会引起合理的愤慨。"
  summary: 一种结构性判断工具。在故事发展过程中，识别出观众"已经知道必须看到"的场景——这是故事承诺的兑现点。从故事前十页的钩子和冲突出发，翻到结尾五到十页，找到必备场景的解决。如果这两处都无法激趣，整个剧本可能不值得投入。必备场景是故事的结构性支点。
  tags: [结构分析, 观众预期, 必备场景, 故事诊断]

- id: f14
  title: 通过对比强调（排版师思维）
  type: framework
  source_chapter: 第 19 章
  source_quote: "强调某个内容，就要淡化处理或排除其他内容。让一个东西抓人眼球，并不总是把它放大那么简单，而是要让它有那么一点与众不同。"
  summary: 一个视觉叙事的决策框架。源自平面设计排版原理：要让A突出，不是把A放大，而是让B、C、D变得不那么引人注目。在电影中，这意味着通过景别对比、灯光对比、声音对比来引导观众注意力。"重要的不是你把阴影放在哪儿，而是你没把阴影放在哪儿。"
  tags: [视觉叙事, 注意力引导, 对比原理, 构图决策]

- id: f15
  title: 视点选择决定主题法
  type: framework
  source_chapter: 第 7 章
  source_quote: "故事视点的选择常常决定着故事主题。剧作家在创作故事时，在早先的版本中选择的是某个主角的视点，但在后来改写时又采用了其他角色的视点，这也很常见。"
  summary: 一个叙事设计的核心原则。选择谁作为叙事视点（谁是主角）就选择了故事的主题。同一事件从不同角色的视角讲述，主题完全改变。《第三人》从莱姆视角讲是"失去的纯真"，从莱姆自己的视角讲则没有角色发展。编剧应先确定主题，再反推视点选择。
  tags: [叙事设计, 视点选择, 主题, 编剧决策]

- id: f16
  title: 背景故事当前化呈现法
  type: framework
  source_chapter: 第 7 章/第 14 章
  source_quote: "背景故事最为有力的呈现方式就是使之成为当前的情节，意即作为先前事件的再现，出现在此时此刻的银幕上。"
  summary: 一种处理背景交代的决策原则。任何背景信息不应通过闪回、画外音或说明性对白呈现，而应转化为当前的戏剧动作。例如《码头风云》出租车内对话：特里讲述过去被利用打拳击的经历——台词是角色说明，但真正的"动作"是这段话对查利产生的当下的、实时的影响。
  tags: [背景交代, 叙事技巧, 剧本修改, 信息呈现]

- id: f17
  title: 虚构空间建构与轴线维护
  type: framework
  source_chapter: 第 20 章/第 24 章
  source_quote: "观众在脑海中构建的'连续空间'其实是导演通过剪辑拼接的碎片。这就是为什么轴线规则如此重要——它维护了这个幻觉的一致性。"
  summary: 一个理解电影空间的思维模型。银幕上的连续空间是观众大脑的幻觉建构，导演通过剪辑拼接碎片图像来维持这一幻觉。180度轴线规则是维护空间一致性的核心契约。越轴会打破观众的方向感，造成混乱——但有意识地打破则可以制造特定效果（如迷失感）。

- id: f18
  title: 编剧修改的奥德兹法则
  type: framework
  source_chapter: 第 14 章
  source_quote: "在人物交锋的场景里，要确保每个人物都带上足够的弹药。聪明的人物在交锋前通常会在脑海中演练若干攻防方案，他们不仅会考虑自己要说什么，还会考虑对方听到后可能会作何反应。"
  summary: 一种对手戏的诊断和增强框架。交锋场景应像棋局或论战：双方都有策略、弹药和后备方案。编剧需明确每方的欲求、障碍、说服手段和威胁。当一方暂时占上风时，另一方亮出"王牌"（未知信息），形成突转。同时引入"反衬人物"（foil）将两人对话升级为多方互动，增加场景密度。
  tags: [对手戏, 场景构建, 对白写作, 剧本修改]

- id: f19
  title: 主角濒危判断法
  type: framework
  source_chapter: 第 16 章
  source_quote: "编剧中有这么一条原理，即主角是我们感觉最濒于险境的那个角色，对于主角而言，支撑整场戏的主要戏剧冲突是他无法逃脱的、必须要面对的一刻。"
  summary: 一个判断主角和冲突强度的工具。在任何场景中，谁最濒于险境、谁面临最不可回避的对抗，谁就是主角。这个标准帮助编剧检验：（1）主角是否真正处于风险中；（2）主要冲突是否足够强烈；（3）角色的主动性是否充分。被动的主角需要更强的外在压力来制造戏剧性。
  tags: [角色判断, 主角确认, 冲突强度, 场景分析]

- id: f20
  title: 搜集-整理-假设-调研循环法
  type: framework
  source_chapter: 第 16 章
  source_quote: "当你在一个文本中寻找新的理解时，做一些'背景调研'可能会有帮助。不要让这个场景本身素材把自己限制住，调研一下更广的背景。"
  summary: 一种从已知材料中挖掘新理解的推理路径。面对创作瓶颈时：（1）先充分理解当前材料；（2）不要局限于文本本身；（3）向外扩展调研更广的背景（百科全书、历史、相关领域）；（4）带着新发现回到文本，往往能找到全新的角度。所罗门王案例中，通过查阅王位继承争议，将一个自信的国王变成了一个急需证明自己的年轻人。
  tags: [调研方法, 创作推理, 研究路径, 写作策略]

- id: f21
  title: 掌握规律后打破规律法
  type: framework
  source_chapter: 前言
  source_quote: "任何'规律'都没有价值，直至你对它有了重新发现。吸收现有的习惯用法，研究它们是如何驾驭了这种媒介的，然后找到使用这些习惯用法的全新手法，这就是创新。"
  summary: 一个关于学习和创新的元认知框架。学习路径为：（1）完全掌握现有规律和惯例；（2）研究规律的例外以理解规律的本质价值；（3）找到用这些惯例表达全新内容的方法——这才是创新。毕加索在重塑视觉语言前已完全掌握写实画。不掌握规律就谈不上真正的创新。
  tags: [学习方法, 创新理论, 元认知, 技艺习得]

- id: f22
  title: 反应重于动作的剪辑原则
  type: framework
  source_chapter: 第 10 章/第 26 章
  source_quote: "编剧的任务是描摹反应而非动作本身。无论拍什么场景，带着自己内在情感和思维的倾听者（或观察者）常常比讲述者或他们正在关注的事件更重要。"
  summary: 电影媒介的本质偏好决定了一个创作原则：观众更想看到的不是动作本身，而是角色对动作的反应。在剪辑中，反应镜头的时机和选择往往比动作镜头更能传达情感深度。这个原则贯穿编剧、导演和剪辑三个阶段：编剧应写出为反应留空间的结构，导演应多拍反应镜头，剪辑师应根据反应来安排节奏。
  tags: [剪辑原则, 表演指导, 编剧策略, 电影本质]

- id: f23
  title: 形式与内容不可分割的统一观
  type: framework
  source_chapter: 导言/第 8 章
  source_quote: "电影语法的技术需求绝不能脱离故事本身来实现，同时，任何一个成功的电影叙事也必然证明了'形式与内容永远无法截然分开'。"

- id: f27
  title: 观众好奇心驱动剪切法
  type: framework
  source_chapter: 第 25 章
  source_quote: "每一个镜头都应自动包含某种元素，它能让观众心中激起跟随见证人在合适的时刻切换到下一个视点的欲望。这就是电影剪辑师所致力于寻找的：切换时空的动机。"
  summary: 剪辑决策的核心原则。每一次剪切的动机都必须预先埋入前一个镜头中。当观众的好奇心被激发（"我想看看他在看什么"），在那个精确时刻切换，剪切就是自然的。如果新画面满足了上一个画面激发的好奇心，或带来意外之喜，即使意外也可接受。反之，没有动机的剪切会让观众困惑。

- id: f28
  title: 悬念与震惊的二择决策
  type: framework
  source_chapter: 第 25 章
  source_quote: "如果是制造悬念，那么必须先让观众看到他们等待的是什么。如果目的是让观众吃惊，那么预警则必须是反向的：必须在意外发生之前刻意将观众引向相反的方向。"
  summary: 面对任何戏剧性事件的呈现方式决策。两个选择：（1）酝酿式——提前告知观众即将发生什么，让观众在等待中体验焦虑（悬念）；（2）突发式——刻意误导观众，让事件出其不意地发生（震惊）。两种方式都需要提前布局——悬念需要预告，震惊需要反向引导。选择取决于故事的整体节奏需求。

- id: f29
  title: 场景必要性检验法
  type: framework
  source_chapter: 第 7 章/第 26 章
  source_quote: "每个入场都是上一个情境的退场，而每个退场又是进入到一个别的什么地方。如果情况和这不一样，那么你就该自问，这场戏是否有存在的必要。"
  summary: 一个场景级别的诊断工具。检验标准：（1）这场戏的开始是否承接上一场戏的结束状态？（2）这场戏的结束是否为下一场戏制造了新的起点？（3）如果从故事中删去这场戏，故事整体是否受损？如果三者都不满足，这场戏应该被删除。"能砍掉的都应该砍掉，剩下的部分则获得了极大的增强。"
  tags: [场景诊断, 必要性检验, 剧本修改, 删减原则]

- id: f30
  title: 先从表演者视角后从画面视角法
  type: framework
  source_chapter: 导言/第 8 章
  source_quote: "导演在考量一个剧本时，首先要做的就是把场景切分成动作……同学们必须要做到首先从表演者的视角去考虑，然后再考虑物体在取景框里看起来怎么样。"
  summary: 导演处理每个场景的思考顺序。第一步：切分为"动作"——标注角色意图、目标和戏剧节拍（表演者视角）。第二步：考虑视觉呈现——摄影机位置、景别、构图（画面视角）。颠倒这个顺序会导致"画面好看但没有戏剧意义"的常见错误。导演的工作始于故事，终于影像。
