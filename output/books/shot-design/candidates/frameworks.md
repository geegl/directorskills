## F01: 故事板快速影像化系统
```yaml
id: F01
title: 故事板快速影像化系统
type: framework
source_chapter: 第三章 故事板
source_quote: "在纸上组织镜头所需的思考过程和心理状态是无价的。"
summary: 用简笔人物图+箭头+立方体的三层叠加法，在无需绘画技巧的情况下快速将剧本转化为可视化故事板。核心是"你省略的部分才是重要的"——简约不是品味而是必要性。
tags: [故事板, 影像化, 快速原型, 简笔画]
```

## F02: I/L/A形态对话调度系统
```yaml
id: F02
title: I/L/A形态对话调度系统
type: framework
source_chapter: 第九章、第十章、第十一章
source_quote: "形态决定摄影机位置，而摄影机位置又是根据动作轴线而来。"
summary: 用I（双人）、L（三人-侧翼）、A（三人-夹击）三种几何形态组织对话场景的演员调度。每种形态可衍生10种基本位置，覆盖从双人到多人对话的全部场景。形态决定摄影机位置，位置决定演员在画面中的站法。
tags: [对话调度, 场面调度, 几何形态, 系统化]
```

## F03: 动作轴线/180°规则系统

```yaml
id: F03

```

## F04: 运动镜头建构单位法
```yaml
id: F04
title: 运动镜头建构单位法
type: framework
source_chapter: 第十二章 运动镜头的调度、第二十章 推轨镜头的调度
source_quote: "不管有多少单个的镜头，都可以被连成一个单一、不间断的镜头。"
summary: 将运动镜头分解为建构单位（基本调度形态），再通过演员运动和摄影机运动将多个单位连接成单一连续镜头。反向运动（counter move）是核心技巧——演员与摄影机朝相反方向运动。
tags: [运动镜头, 建构单位, 长镜头, 调度设计]
```

## F05: 视点认同度谱系
```yaml
id: F05
title: 视点认同度谱系
type: framework
source_chapter: 第十六章 视点
source_quote: "每个镜头都有它主观性的程度，更精确地说，都提供了我们认同的程度。"
summary: 从全知视点→受限制的第三人称→第一人称主观的认同度连续谱。摄影机越靠近特写中演员的视线，观众认同度越高。包含影像控制（构图与调度）和叙事控制（情节跟随）两种操控方法。
tags: [视点, 认同度, 主观镜头, 叙事控制]
```

## F06: 开放与封闭构图策略
```yaml
id: F06
title: 开放与封闭构图策略
type: framework
source_chapter: 第十五章 开放与封闭式构图
source_quote: "开放式构图借着构图与故事元素的使用，创造了一种我们得以分享男人心理活动的脉络。"
summary: 开放式构图使用前景元素涵盖主体，将观众带入场景空间，创造心理认同。封闭式构图保持自足构图，维持情感距离。两种方法通过版本迭代法（同一场景设计多个版本）来探索不同的戏剧效果。
tags: [构图, 开放式构图, 封闭式构图, 版本迭代]
```

## F08: 戏剧动作范围三区间法
```yaml
id: F08
title: 戏剧动作范围三区间法
type: framework
source_chapter: 第十三章 画面的纵深
source_quote: "每种场景中动作的戏剧性范围，是由动作空间的面积和形状所决定的。"
summary: 将摄影机前空间分为前景/中景/后景三个区间，摄影机可在动作范围之内或之外拍摄。纵深调度（B方式）允许单一机位覆盖场景，压缩调度（远摄镜头）则强调横向运动。威尔斯用空间分隔角色，雷诺阿用空间让角色相交。
tags: [纵深调度, 景深, 空间组织, 威尔斯vs雷诺阿]
```
