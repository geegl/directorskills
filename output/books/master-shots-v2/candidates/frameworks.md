title: 力量对比三要素组合框架
type: framework
source_chapter: "第一章 1.1 力量对比"
source_quote: "要把三种对比结合起来，才能达到最终的效果。一台相机水平拍摄，一台仰拍；一台采用短焦镜头，一台长焦；一台低角度拍摄，一台与演员头同高。"
summary: >
  通过三个维度的对比同时叠加来建立角色间的力量差异：(1)摄影机角度（仰拍vs平拍），(2)镜头焦距（短焦压迫感vs长焦亲和感），(3)摄影机高度（低角度强势vs等高平等）。三者缺一不可——单独使用任何一个维度都无法产生足够的力量对比效果。导演在拍摄对话中的权力关系时，应将这三个变量视为一个组合工具包，同时在A角色身上施加"强势三件套"（仰拍+短焦+低角度），在B角色身上施加"弱势三件套"（平拍+长焦+等高）。
tags: [power-dynamics, camera-angle, focal-length, camera-height, contrast, dialogue]

id: f03
title: 180度线破规则框架
type: framework
source_chapter: "第一章 1.4 跨越界线"
source_quote: "在两名演员之间画一条线，在一般情况下，你应该让摄影机保持在线的一边...如果你能在正确的时间小心地打破这些规则，那你就能制造出冲突感。"
summary: >
  常规规则要求摄影机始终在轴线同一侧以保持空间连续性。但当导演需要制造迷失方向感、紧张或争吵情绪时，可以故意跨越轴线。关键执行条件：(1)角色必须有合理的背对对方的理由（如等电梯），让跨线看起来自然；(2)镜头不断在两侧之间转换，制造混乱感但不完全丧失方向感——观众仍能通过角色对话理解关系；(3)整体节奏应配合争吵的急促感。这将一个"大忌"转化为有力的情绪工具。
tags: [180-degree-rule, rule-breaking, spatial-orientation, tension, conflict]

title: 环拍静止对比框架
type: framework
source_chapter: "第二章 2.1 环拍对话"
source_quote: "一直以较远的距离环拍，导演便制造出一种角色静止的效果...总有一个瞬间，观众能直接注视到其中一个角色的眼睛。"
summary: >
  当对话表面平静但内在紧张时，让摄影机环绕拍摄静止的演员。环拍产生三重效果：(1)延长时间，给密集对话增加视觉冲击；(2)制造角色静止不动的错觉——他们在礼貌地轻声耳语；(3)在环拍的某个瞬间，观众会直视到角色眼中的恐惧。关键执行要求：保持摄影机与演员等距且移动平滑——需使用轨道车而非手持，因为手持无法保证距离恒定。该框架将"运动的摄影机"与"静止的演员"之间的张力转化为叙事工具。
tags: [camera-movement, circling, tension-under-calm, orbital-shot, suspense]

id: f08
title: 镜头递进接近框架
type: framework
source_chapter: "第二章 2.2 缩近距离"
source_quote: "开场用了广角镜头进行拍摄，当演员越靠越近的时候，镜头的焦距也增长了。当演员走到一块儿时，再换上最长焦距的镜头。"
summary: >

tags: [focal-length-progression, distance, emotional-journey, lens-selection, technique]

summary: >

id: f16
title: 演员任务过载本能框架
type: framework
source_chapter: "结语"
source_quote: "当你给演员安排越多的任务时，他们的表现反而往往越出色...你得给演员提更多的表演要求，直到他们的脑子转不过来为止。那他们就只能凭本能去表演了。"

  当演员担心自己演得不好时，最好的解决方案不是安慰他们，而是给他们更多任务（走位、手部动作、头部角度、与道具互动等），直到他们的意识没有余力去自我怀疑。过多的自我意识会毁掉表演；当演员被迫同时处理台词、表演、提示和走位时，他们的大脑"转不过来"，只能凭本能表演。本能的表演真实生动，远胜于刻意琢磨。导演在提要求时要和颜悦色但坚决——演员说"这么做不对"往往是因为心慌而非真的不对。该框架将"演员的抵抗"重新解读为"接近突破的信号"。
