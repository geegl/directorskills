# principles.md — 原则提取候选

## P01: 视线剪切原则
```yaml
id: P01
title: 视线剪切原则
type: principle
source_chapter: 第六章、第十六章
source_quote: "让-吕克·戈达尔曾说过，最自然的剪辑是'视线剪切'。"
summary: 眼睛是特写中最具表现力的器官，视线方向定义了场景中的空间关系。观众对视线不协调特别敏感。视线剪切是建立主观镜头和空间连续性的基础工具——先看，再看所看之物。
tags: [剪辑, 视线, 特写, 空间连续性]
```

## P02: 景别比例变化原则
```yaml
id: P02
title: 景别比例变化原则
type: principle
source_chapter: 第六章、第八章
source_quote: "镜头间景别的变化会因情况而异，而且是由我们的认定来决定的。"
summary: 全景、中景、特写是互相参照的比例关系，而非绝对尺寸。景别间的视觉辨识和叙事逻辑共同创造连续空间感。当代观众已能接受比过去更剧烈的景别跳转。
tags: [景别, 比例关系, 连续性, 视觉辨识]
```

## P03: 演员运动动机原则
```yaml
id: P03
title: 演员运动动机原则
type: principle
source_chapter: 第十二章
source_quote: "演员运动的唯一标准就是有无动机，这类调度隶属于舞台动作的范畴。"
summary: 运动调度中演员的走位必须有动机——点烟、穿衣、看地图等日常行为提供自然的运动理由。不要为增加场景内容而人为发明动作。创意应在正确观察人类行为的前提下浮现。
tags: [调度, 动机, 自然主义, 行为观察]
```

## P04: 面向观众原则
```yaml
id: P04
title: 面向观众原则（Frontality）
type: principle
source_chapter: 第九章、第十章
source_quote: "眼睛被清楚呈现的演员，将主宰该镜头。"
summary: 西方艺术传统偏爱将人脸稍微偏离中心放置以避免过分匀称。在对话调度中，面向摄影机的演员获得更强的观众认同和权威感。纵深调度中近处演员通常更强有力，但可通过灯光和叙事脉络转移。
tags: [构图, 认同, 权力关系, 表演]
```

## P05: 运动镜头三分层原则
```yaml
id: P05
title: 运动镜头三分层原则
type: principle
source_chapter: 第十三章、第十九章
source_quote: "使用三个层面的运动，可以加强深度感。"
summary: 运动镜头穿过前景、中景、后景三个运动层面时获得最大深度。摄影机与主体同速时只有两层运动（后景+主体），不同速时增加前景层变为三层。透视感因运动而强化。
tags: [运动镜头, 深度感, 透视, 三分层]
```

## P06: 连续性构图原则
```yaml
id: P06
title: 连续性构图原则
type: principle
source_chapter: 第六章
source_quote: "构图不是以单张图的方式来判断的，而是以它前后连接的关系来判断的。"
summary: 电影画面的平衡与否要根据它与前后画面的关系来判断。偏离中心的特写创造左右移动的视线变化节奏。银幕越宽，偏离中心的构图越普遍。这是剪辑风格的核心——连续性艺术。
tags: [构图, 连续性, 节奏, 银幕宽高比]
```

## P07: 摇镜引导注意力原则
```yaml
id: P07
title: 摇镜引导注意力原则
type: principle
source_chapter: 第十七章
source_quote: "摇镜是那么一种引人注目的运动，以至于我们通常不会质疑它的效果。"
summary: 摇镜自动地暗示两个兴趣焦点之间的逻辑关系。任何运动的物体（汽车、树叶）都可以成为摇镜的叙事诱饵。交叉摇镜具有舞蹈设计般的特质。摇镜的执行能清楚让观众了解两种画面元素之间关系的重要性。
tags: [摇镜, 注意力引导, 叙事连接, 交叉摇镜]
```

## P08: 升降镜头威严感原则
```yaml
id: P08
title: 升降镜头威严感原则
type: principle
source_chapter: 第十八章
source_quote: "升降镜头天生就具有一种威严感，不论主体是什么，单靠它外在运动的快感，就能抓住我们的兴趣。"
summary: 升降镜头是最不自然的摄影机运动（我们很少从高耸位置看事物），因此天然产生威严感和异样感。用于段落开头如"很久很久以前"般引导注意力。希区柯克用它从天花板降到钥匙特写（《美人计》），贝托鲁奇用它表达痛楚（《巴黎最后的探戈》）。
tags: [升降镜头, 威严感, 交代镜头, 情感表达]
```

## P09: 轴线越轴三种方法
```yaml
id: P09
title: 轴线越轴三种方法
type: principle
source_chapter: 第六章
source_quote: "实际拍摄时我们很少为了建立一条新轴线，而进行一番繁复的场面调度和逻辑分析。"
summary: 越轴的三种合法方法：(1)主体越轴并以新运动方向建立新轴线；(2)摄影机越轴跟拍主体至新地点；(3)新主体进入画面建立与原轴线相对的新轴线。摄影机越靠近轴线，越轴越难被发现。
tags: [轴线, 越轴, 空间连续性, 剪辑规则]
```

## P10: 特写亲密性原则
```yaml
id: P10
title: 特写亲密性与隐私权原则
type: principle
source_chapter: 第六章
source_quote: "特写不仅能显露一种亲密关系，也可以让我们感觉像是正闯入一段私人时光。"
summary: 特写给观众带来比日常生活中与任何人更亲密的关系。过度使用会侵犯隐私权——摄影机不需获取被摄者同意，特别是远摄镜头时。每个文化系统对隐私和肢体接触有不同的习俗，电影人可用摄影机操控这种社会距离。
tags: [特写, 亲密感, 隐私, 文化差异]
```

## P11: 摄影角度上下文依赖原则
```yaml
id: P11
title: 摄影角度的上下文依赖性
type: principle
source_chapter: 第十四章
source_quote: "摄影角度本身是没有意义的……一个镜头的价值，乃是由故事的本身来决定的。"
summary: 低角度=强势、高角度=弱势的还原分析法只在某些情况有效。《公民凯恩》中低角度在前期表现凯恩的主宰，在后期反而突显他的渺小——同一个技巧在不同故事语境中产生相反效果。
tags: [摄影角度, 上下文, 公民凯恩, 打破规则]
```
