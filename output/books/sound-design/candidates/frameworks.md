```yaml
  提示单，类似乐谱，标注每个声音的进入方式（淡入/硬切）、位置和处理方式。
tags: [framework, sound-map, iteration, workflow, documentation]
```

```yaml
id: FW-04
title: 四种听音模式 (Four Listening Modes)
type: framework
source_chapter: 第四章
source_quote: "米歇尔·琼对三种听音模式作了辨析：简化模式、因果模式和语义模式。这里还介绍了第四种：参考模式"
summary: >
  听音分为四种模式：1）简化模式——关注声音本身属性（音调/音色/节奏）；
  2）因果模式——通过声音识别声源；3）语义模式——理解语言含义；
  4）参考模式——基于上下文理解声音的情绪和叙事含义。声音设计师需
  在不同模式间切换，理解观众可能的听音角度。
tags: [framework, perception, listening-modes, psychoacoustics, chion]
```

```yaml
source_quote: "声音的主要属性可以分成节奏、强度、音调、音色、声音的发声速度、音型、组织状态等类别"

  声音可按七个维度分类：节奏（有/无节奏）、强度（轻-响/分贝）、
  音调（低-高/Hz）、音色（纯音-噪声/谐波）、速度（发声快慢）、
  音型（音头-音体-音尾包络）、组织状态（有序-无序）。
  这为声音设计师之间的沟通提供了共同语言。
tags: [framework, sound-attributes, acoustics, classification, psychoacoustics]
```

```yaml
id: FW-06
title: 格式塔听觉原理 (Gestalt Principles for Sound)
type: framework
source_chapter: 第四章
source_quote: "格式塔心理学采用了视觉感知相关的术语，这些术语还具有听觉感知上与其相对应的术语"
summary: >
  格式塔原理在听觉中的应用：轮廓与背景（figure/ground）——从环境声中
  突出关键声音；完整性与闭合（closure）——大脑补全缺失的声音信息；
  邻近度和相似度——时间和频率相近的声音被分组；共同命运——同时变化的
  声音被视为整体；幻觉——大脑创造实际不存在的声音感知。
tags: [framework, gestalt, perception, psychoacoustics, figure-ground]
```

```yaml
id: FW-08
title: 听觉层级体系 (Auditory Hierarchy)
type: framework
source_chapter: 第八章
source_quote: "声音支配注意力的三个层级：1）有意识倾听的现场感觉；2）听到但不直接注意的辅助声音；3）影响潜意识的背景声音"
summary: >
  电影声音按注意力分为三层：前景层（有意识关注，如对白）、
  中景层（听到但不直接注意，如环境声）、背景层（无意识但影响情绪，
  如低频氛围）。层级之间可相互转换。语言通常为最高层级，
  音乐和音效需围绕语言信息传达进行调整。
tags: [framework, auditory-hierarchy, attention, mixing, figure-ground]
```

```yaml
id: FW-09

type: framework
source_chapter: 第五章/第八章
source_quote: "奏鸣曲形式：呈示部（两个主题）→展开部（主题变换/对立）→再现部（回归/解决），与英雄之旅结构相对应"
summary: >

  呈示部=第一幕/出发（建立主题/角色），展开部=第二幕/冒险
  （冲突/变换/对立），再现部=第三幕/回归（解决/融合）。
  这为声音设计提供了非语言的叙事结构蓝图。
tags: [framework, sonata-form, hero-journey, three-act, dramatic-structure]
```

```yaml
id: FW-10
title: 声画关系分类体系 (Sound-Image Relationship Taxonomy)
type: framework
source_chapter: 第七章
source_quote: "画内声音（看得到声源）/画外声音（看不到声源但在故事中存在）/非剧情声音（故事人物听不到）"
summary: >
  电影声音按与画面的关系分为：画内声音（声源可见，如嘴唇同步的对白）、
  画外声音（声源不可见但在故事空间内，可主动引发好奇或被动营造氛围）、
  非剧情声音（人物听不到，如画外音、配乐）。还有"幻听"（acousmatic）
  概念——听不到声源的声音，以及"去幻听"的戏剧揭示。
tags: [framework, diegetic, non-diegetic, on-screen, off-screen, acousmatic]
```
