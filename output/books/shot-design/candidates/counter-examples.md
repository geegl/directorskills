# counter-examples.md — 常见错误与反例提取

## CE01: 在轴线错误侧拍摄导致空间混乱
```yaml
id: CE01
title: 跳轴导致银幕方向颠倒
type: counter-example
source_chapter: 第六章
source_quote: "当剪辑在一起时，从摄影机A和B所得镜头的银幕方向将会相反。"
summary: 当两个摄影机都位于动作轴线上拍摄时，剪辑在一起会产生银幕方向颠倒。1960年前这是严格禁止的，但现代观众已能理解这种剪辑模式。关键判断：人物以侧面呈现时颠倒最突兀，正面/背面拍摄时帮助分辨。
tags: [跳轴, 银幕方向, 空间混乱, 剪辑错误]
```

## CE02: 远摄镜头消灭调度深度
```yaml
id: CE02
title: 远摄镜头的调度陷阱
type: counter-example
source_chapter: 第十三章
source_quote: "远摄镜头的趋势仍然持续着……使得一般人的电影知识素养和对剪辑、走位变化的认识，在过去的十年中走向衰微。"
summary: 远摄镜头因消除了许多动作调度和剪辑问题而受欢迎，但它的吸引力多半是图像上的。远摄镜头强调横向运动，使纵深调度难以区分形态差异。运动调度中，只有横向运动才能有效强调重要动作。过度依赖远摄镜头导致调度能力退化。
tags: [远摄镜头, 调度退化, 浅景深, 偷懒]
```

## CE03: 为运动而运动——无人物动机的调度
```yaml
id: CE03
title: 无人物动机的虚假运动
type: counter-example
source_chapter: 第十二章
source_quote: "你绝不需要只是为了增加场景的内容，而人为地发明动作。"
summary: 常见错误是为了增加场景内容而人为发明演员走位。运动调度的标准是"有无动机"——演员的走位必须来自自然行为（点烟、穿衣、看地图），而非导演的刻意安排。如果故事、演员和导演的进程配合无间，创意会在正确观察人类行为的前提下浮现。
tags: [虚假调度, 动机缺失, 人为运动, 表演]
```

## CE04: 特写滥用——侵犯隐私权
```yaml
id: CE04
title: 特写的过度使用与隐私侵犯
type: counter-example
source_chapter: 第六章
source_quote: "有时候人们会过度使用这种检视的能力，这样的话，特写就通过强迫拉近彼此的亲密关系侵犯了隐私权。"
summary: 特写可以拉近观众与动作的距离，但过度使用会通过强迫亲密关系侵犯隐私权。摄影机不需获取被摄者同意，特别是远摄镜头时。电视新闻摄影师常在人们遭难时以大特写揭人隐私，观众可能感到不悦。
tags: [特写滥用, 隐私, 亲密距离, 伦理]
```

## CE05: 开放/封闭构图混用导致观众混乱
```yaml
id: CE05
title: 开放与封闭构图的不一致使用
type: counter-example
source_chapter: 第十五章
source_quote: "全部使用开放或封闭式构图策略是少见的。"
summary: 在一部片子里全部使用开放或封闭式构图策略是少见的。常见错误是在同一场景中不一致地混用两种策略，导致观众无法建立稳定的情感距离。正确做法是根据戏剧需要在两种策略间有意识地切换，并保持场景内部的一致性。
tags: [构图不一致, 情感距离, 风格混乱, 观众体验]
```

## CE06: 前景元素遮挡关键信息
```yaml
id: CE06
title: 前景遮挡与分像屈光度透镜的误用
type: counter-example
source_chapter: 第十三章
source_quote: "分像屈光度透镜只能在摄影机将前景和后景元素锁定在画面一半的区域里时使用。"
summary: 分像屈光度透镜在效果产生的边缘有一条可见的模糊线，必须通过对镜头中所有元素技巧性的构图来隐藏。人物或对象都不可以跨越这条线。常见误用是让人物跨越分割线导致可见的模糊区域暴露。
tags: [分像屈光度透镜, 前景遮挡, 技术限制, 构图]
```

## CE07: 运动镜头与固定镜头的突兀剪接
```yaml
id: CE07
title: 运动镜头与固定镜头的剪辑困难
type: counter-example
source_chapter: 第十九章
source_quote: "我们很难在一个推轨镜头中，安插一个以固定镜头拍摄的相同动作的镜头。"
summary: 常见错误是在推轨镜头中插入相同动作的固定镜头，造成视觉断裂。运动镜头能为下一个段落镜头的影像复杂性和变化做预备，使影像转移更平顺。当衔接两个运动镜头时，跳接显得不那么明显。
tags: [运动镜头, 剪辑断裂, 节奏, 跳接]
```

## CE08: 演员面部特写中景别与空间统一性
```yaml
id: CE08
title: 特写破坏空间统一性
type: counter-example
source_chapter: 第九章、第十章
source_quote: "它容易分割空间；在同一个镜头里，无法同时看见说话的以及在倾听的演员。"
summary: 在对话场景中过度使用单人特写会分割空间，使观众无法同时看到说话者和倾听者的反应。过肩镜头能统一空间，在交代镜头之后，整场对话可以很轻松地用过肩镜头处理，偶尔使用三人镜头或双人镜头来变化节奏。
tags: [特写分割, 空间统一, 过肩镜头, 对话场景]
```
