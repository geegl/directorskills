```yaml
id: p01
title: "肤色在矢量示波器上的楔形范围原则"
type: principle
source_chapter: 第8章
source_quote: "肤色有可能出现的所有色调范围被限制在很明显的一簇波形中，在这个范围内波形仍有足够的变化空间。"
summary: 所有人种的肤色在矢量示波器上都落在一个狭窄的楔形范围内（围绕I-bar/肤色线）。RGB分量示波器上肤色总是红色通道最强、绿色通道较弱、蓝色通道最弱。肤色差异主要体现在亮度而非色相。调色师应利用这一物理事实作为色彩平衡的基准点。
tags: [肤色, 矢量示波器, I-bar, 色彩平衡基准]
```

```yaml
id: p02
title: "先对比度后色彩的调整顺序原则"
type: principle
source_chapter: 第4章
source_quote: "三色平衡控件对图像灰阶区域的影响依赖于亮度通道，所以，对亮度通道的调整会相应影响色彩调整的结果。"
summary: 调色应先调整对比度再调整色彩，因为色彩平衡控件的影调分区（暗部/中间调/高光）依赖于亮度通道。改变对比度会改变各影调区的范围，从而影响色彩调整的效果。反之，先调色彩再调对比度会导致色彩偏移。这是一个强交互关系，需要迭代调整。
tags: [调整顺序, 对比度, 色彩平衡, 交互性]
```

```yaml
id: p03
title: "色温决定场景感知色彩"
type: principle
source_chapter: 第4章
source_quote: "色温，是调色师需要理解的概念中最重要之一。因为在任何一个场景中，色温的变化，都会改变观众的感知色彩。"
summary: 色温（开尔文）描述光源的颜色特质。温度越高光越蓝（日光6500K），越低越红（烛光1500K）。调色师在调色时本质上是在控制光源色温。色温变化会改变观众对场景中所有物体颜色的感知。混合色温（如钨丝灯+日光）会造成画面内不同区域偏色不同，需要分区处理。
tags: [色温, 开尔文, 光源, 感知色彩]
```

```yaml
id: p04
title: "对比度与饱和度的联动效应"
type: principle
source_chapter: 第3章
source_quote: "使用常规的反差工具（Lift, Gamma, Gain）来扩大对比度，导致了图像的饱和度提升。"
summary: 在RGB处理方式下，扩大对比度会自动增加饱和度（因为色度信号随亮度变化被放大），压缩对比度则降低饱和度。在Y-only（仅亮度）模式下调整对比度则不影响测量饱和度但会改变感知饱和度。调色师需要理解这种联动关系，在调对比度后检查并补偿饱和度变化。
tags: [对比度, 饱和度, 联动效应, RGB处理]
```

```yaml
id: p05
title: "记忆色彩三大要素：肤色、天空、植物"
type: principle
source_chapter: 第8章
source_quote: "这三个记忆色彩尤其反复地出现在众多研究中，大多数观众对这组记忆色彩表现出明显的偏好。"
summary: 研究反复证实，观众对三种物体有强烈且一致的色彩记忆偏好：(1)肤色——比实际偏黄/金色；(2)天空——比实际更青色、饱和度更高；(3)植物/草地——比实际更蓝绿色。这些记忆色彩是调色师进行二级调色时最常被隔离调整的三个元素，也是客户最先提出修改要求的对象。
tags: [记忆色彩, 肤色, 天空, 植物, 观众偏好]
```

```yaml
id: p06
title: "高光感知与阴影深度的相对性原则"
type: principle
source_chapter: 第3章
source_quote: "对高光的感知与阴影的深度有关，这点极为重要，需要记住。由于人眼的工作原理，观众对高光的感知与图像的暗部密切相关。"
summary: 观众对高光亮度的感知是相对于暗部深度的。有时压低阴影而保持高光不动，比提高高光更能有效增强对比度。图像的对比度完全是相对的——通过改变参照系（暗部深度）可以改变对同一亮度值的感知。这一原则适用于所有对比度调整决策。
tags: [感知相对性, 对比度, 高光, 阴影, 人眼视觉]
```

```yaml
id: p07
title: "日落/晨光的三区色彩特征"
type: principle
source_chapter: 第8章
source_quote: "非常温暖的高光偏色取决于太阳的位置。中间调的暖光。接近正常颜色的暗部。"
summary: 黄金小时的光照有三个关键特征：(1)高光极度偏暖（橙/红色），色温可低至1600K；(2)中间调也偏暖但强度弱于高光，因为来自天空散射光；(3)暗部接近正常颜色，因为主要受环境反射光影响。调色师模拟日落效果时应遵循这三个层次——暖高光、温和暖中间调、干净暗部。
tags: [日落, 晨光, 黄金小时, 色温分层, 模拟光照]
```

```yaml
id: p08
title: "光学滤镜的非线性着色特性"
type: principle
source_chapter: 第11章
source_quote: "光的着色强度会非线性地应用在整个图像的影调范围内。图像较亮部分更易受滤镜的影响，而图像较暗部分则受其影响较小。"
summary: 光学滤镜（如雷登85C）对图像的影响是非线性的：高光受影响最大，中间调次之，暗部几乎不受影响。这意味着在数字调色中模拟真实滤镜效果时，不应简单地对全画面均匀加色，而应让着色强度随亮度变化。用曲线工具可以精确模拟这种非线性着色。
tags: [滤镜, 非线性着色, 高光, 数字模拟]
```

```yaml
id: p09
title: "镜头匹配中的饱和度独立评估原则"
type: principle
source_chapter: 第9章
source_quote: "图像的色彩平衡是正确的但饱和度不够。大家很容易会直接调整色彩平衡来创建匹配，但实际上，色彩平衡其实可能会导致饱和度提升。"
summary: 镜头匹配时，必须将饱和度作为独立维度来评估。常见错误：两个镜头色彩平衡匹配了但饱和度不同，调色师误以为需要更多色彩平衡调整，实际上只需统一饱和度即可。学会用肉眼分辨细微饱和度差异比依赖示波器更重要。
tags: [镜头匹配, 饱和度, 独立评估, 常见错误]
```

```yaml
id: p10
title: "形状遮罩保持简单原则"
type: principle
source_chapter: 第6章
source_quote: "如果是调整亮度、暗部，或改变色温，简单的形状通常比复杂的形状更可取、更好用。"
summary: 形状遮罩应尽可能保持简单。简单的椭圆/矩形比复杂的贝塞尔曲线更快创建、更容易调整、关键帧动画也更省时。只有在需要精确隔离特定物体轮廓时才使用复杂形状。遮罩的目的是分离光影区域而非精确勾勒物体轮廓——观众不会看到遮罩本身。
tags: [形状遮罩, 简单原则, 效率, Power Window]
```
