## Book Overview

**Core thesis:** 对话场景不应使用标准的"正反角度拍摄"，而应通过精心设计的演员走位与摄影机运动轨迹的配合（"编舞法"），将对话的情感、权力关系和戏剧冲突可视化。

**Skill count:** 14 (12 frameworks + 2 principles) extracted from 130 candidates (38% pass rate through triple verification)

**Domain:** 镜头域 (Shot) — 对话场景镜头设计与演员调度

## Usage Guide: Which Skill for Which Problem

### 🎭 "对话场景看起来假/不自然"

| Problem | Primary Skill | Supporting Skills |
|---------|--------------|-------------------|
| 演员对视不自然，像在演话剧 | **v12** 给演员找事干+避免对视 | v07 任务过载 |
| 演员说"走位不对"但说不出原因 | **v07** 任务过载触发本能表演 | v12 找事干+不对视 |
| 对话场景太平淡没有活力 | **v12** + **v07** 组合使用 | v05 位置交换 |

### 📐 "空间/构图问题"

| Problem | Primary Skill | Supporting Skills |
|---------|--------------|-------------------|
| 房间太小拍不出空间感 | **v02** 深布景三层空间 | v01 障碍物叙事 |
| 多人对话观众不知道谁在哪 | **v08** 目光方向空间映射 | v06 摄影机高度 |
| 两人之间有隔阂需要可视化 | **v01** 障碍物叙事 | v05 位置交换 |
| 权力关系反转不用台词表现 | **v05** 位置交换象征权力转移 | v06 摄影机高度 |

### 🎥 "摄影机该放哪/跟谁"

| Problem | Primary Skill | Supporting Skills |
|---------|--------------|-------------------|
| 观众应该站在谁的立场 | **v06** 摄影机高度即立场 | v10 摄影机跟随 |
| 追逐场景该跟追的人还是被追的人 | **v10** 摄影机跟随即认同 | v06 高度即立场 |
| 主角出场不够震撼 | **v03** 延迟揭示主角 | v09 特写时机 |

### 🔭 "焦距/镜头选择"

| Problem | Primary Skill | Supporting Skills |
|---------|--------------|-------------------|
| 同一角色不同场景要显得更友好/更可怕 | **v13** 焦距决定角色气质 | v06 摄影机高度 |
| 两人走近的过程太平淡 | **v04** 焦距递进增强接近 | v13 焦距气质 |
| 角色听到坏消息怎么拍 | **v14** 幽闭空间长焦压迫 | v09 特写时机 |

### ✂️ "什么时候切镜头"

| Problem | Primary Skill | Supporting Skills |
|---------|--------------|-------------------|
| 对话场景该不该切特写 | **v09** 特写时机选择 | v08 目光映射 |
| 角色崩溃后怎么回到日常 | **v11** 距离坍缩与退出叙事弧 | v09 特写时机 |

### Quick Decision Tree

```
你的对话场景有什么问题？
│
├─ "看起来不真实" → v12 (找事干+不对视)
│   └─ 还是不行？ → v07 (任务过载)
│
├─ "观众搞不清关系/位置"
│   ├─ 两人 → v01 (障碍物) 或 v05 (位置交换)
│   └─ 三人以上 → v08 (目光映射)
│
├─ "缺乏视觉冲击力"
│   ├─ 主角出场 → v03 (延迟揭示)
│   ├─ 权力反转 → v05 (位置交换) + v06 (高度)
│   ├─ 情感爆发 → v14 (长焦压迫) 或 v09 (特写时机)
│   └─ 关系变化 → v04 (焦距递进)
│
├─ "不知道摄影机怎么动"
│   ├─ 跟谁？ → v10 (跟随=认同)
│   ├─ 多高？ → v06 (高度=立场)
│   ├─ 推近还是退出？ → v11 (距离坍缩弧线)
│   └─ 空间太小？ → v02 (深布景)
│
└─ "不知道用什么焦距"
    ├─ 角色气质 → v13 (焦距=气质)
    └─ 接近/疏远 → v04 (焦距递进)
```
