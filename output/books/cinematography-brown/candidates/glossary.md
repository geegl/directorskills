# Glossary — 电影摄影：理论与实践

---

```yaml
id: g01
title: 主光（Key Light）
type: glossary
source_chapter: 第7章 布光基础
source_quote: "主光：打在人物或物体上的关键光线，是场景中的'主要'灯光。"
summary: 主光是场景中最主要的光源，决定了画面的主要照明方向、阴影位置和整体明暗分布。主光的方向（前/侧/背）是布光中最关键的决策，直接影响场景的情绪和基调。
tags: [灯光术语, 主光, 布光基础]
```

```yaml
id: g02
title: 补光（Fill Light）
type: glossary
source_chapter: 第7章 布光基础
source_quote: "补充光：照亮主光照不到的阴影部分的光线，场景的灯光有时会用主/补光比作为评价标准。"
summary: 补光用于控制主光产生的阴影深度。主/补光比（灯光比）是衡量场景对比度的核心指标。高比值=阴影深重（低调光），低比值=阴影浅淡（高调光）。补光通常比主光柔和，可以来自反光板、柔光灯或环境反射。
tags: [灯光术语, 补光, 灯光比, 对比度]
```

```yaml
id: g03
title: 背光/轮廓光（Back Light / Rim Light）
type: glossary
source_chapter: 第7章 布光基础
source_quote: "背光：从人物或物体背后射来的光，会在主体边缘形成一圈亮边，令人脸的暗部从深色背景中分隔出来。"
summary: 背光从主体后方照射，在主体边缘形成一圈亮边（轮廓光），将主体从背景中分离出来。背光可以过曝，在胶片和录像上仍然成像良好，因此有时被称为"发型光"或"肩后光"。背光是三点布光中增加立体感的关键元素。
tags: [灯光术语, 背光, 轮廓光, 分离感]
```

```yaml
id: g04
title: 强聚光（Kicker）
type: glossary
source_chapter: 第7章 布光基础
source_quote: "强聚光是从演员后方射向他脸庞暗部（与主光方向相反的部分）的光线。强聚光通常能很好地照亮脸部，甚至不需要再加入补光了。"
summary: 强聚光是从背后3/4角度照射在主体脸颊暗部的光线，不要与背光混淆（背光照亮的是物体两边）。强聚光特别有用：即使完全没有补光，它也能很好地照亮脸庞暗部。对男性常用较硬的强聚光，对女性常用较软的。
tags: [灯光术语, 强聚光, 脸部照明, 布光技巧]
```

```yaml
id: g05
title: T-stop（真正档值）
type: glossary
source_chapter: 第14章 光学和对焦
source_quote: "T档（真正档值）是在光具座上测量出来的镜头实际传播光量。计算景深和超焦时需要用到f/档值，而设置曝光时会用到T档值。"
summary: T-stop是镜头实际透光率的精确测量值，与f-stop（理论进光量）不同。f-stop是焦距与光圈直径之比（数学计算），T-stop考虑了镜片组的光损失（实际测量）。设置曝光用T-stop，计算景深用f-stop。不同镜头即使f-stop相同，T-stop也可能不同。
tags: [光学术语, T-stop, 曝光, 镜头]
```

```yaml
id: g06
title: 景深（Depth of Field）
type: glossary
source_chapter: 第14章 光学和对焦
source_quote: "通过调小光圈，或换上更加广角的镜头，我们可以将他们也纳入焦点内。只有在图像平面上清晰投影的物体才是进入了'临界焦点'。"
summary: 景深是指在焦点平面前后仍被认为"清晰"的范围。影响景深的因素：光圈大小（小光圈=大景深）、焦距（广角=大景深）、物距（远=大景深）、传感器尺寸（小传感器=大景深）。景深是重要的叙事工具：浅景深隔离主体（亲密感），深景深包含环境（信息量大）。
tags: [光学术语, 景深, 叙事工具, 焦点]
```

```yaml
id: g07
title: 色温（Color Temperature）
type: glossary
source_chapter: 第12章 色彩
source_quote: "色温的单位是开尔文度，简称大K。色温超过5000K是冷色（略带蓝色的白色），而更低的色温（2700—3000K）叫做暖色。"
summary: 色温用开尔文（K）衡量光源的冷暖特性。标准参考点：烛光≈1900K，钨丝灯≈3200K，日光≈5500K，阴天≈6500K，蓝天≈10000K+。色温是光源的物理属性，而"暖色/冷色"是人对色彩的心理反应——红色/橙色感觉温暖，蓝色感觉冰冷。
tags: [色彩术语, 色温, 开尔文, 光源特性]
```

```yaml
id: g08
title: 显色指数（CRI - Color Rendering Index）
type: glossary
source_chapter: 第12章 色彩
source_quote: "对于摄影来说，只有显色指数在90以上的光源是可以使用的。"
summary: 显色指数衡量光源还原物体真实颜色的能力，范围0-100。CRI 90以上才适合摄影使用。荧光灯和工业光源（钠蒸汽灯、汞蒸汽灯）通常CRI很低，因为它们的光谱不连续。CRI低的光源无法仅通过滤片完全矫正，需要替换灯管或使用减绿片/CC滤片组合。
tags: [色彩术语, 显色指数, 光源质量, 荧光灯]
```

```yaml
id: g09
title: 减光器（Dimmer）
type: glossary
source_chapter: 第7章 布光基础, 第8章 光源
source_quote: "每一个真实光都必须装一个减光器来进行光线微调，用于这个目的的小型减光器叫作手动调光器。"
summary: 减光器用于无级调节灯光亮度。每个场景中的真实光源（台灯、落地灯等）都应配备减光器以便微调。减光器的注意事项：钨丝灯减光时色温会降低（变暖变红），HMI灯不能使用普通减光器（需要专用电子减光器），LED灯减光不影响色温。
tags: [设备术语, 减光器, 灯光控制, 真实光源]
```

```yaml
id: g10
title: 示波器（Waveform Monitor）
type: glossary
source_chapter: 第9章 高清摄影
source_quote: "示波器显示的是录像信号中的明度信息，你能通过它分析整个画面或单根扫描线的信息。0表示纯黑，100表示纯白。"
summary: 示波器是录像拍摄中替代测光表的工具，显示画面的亮度分布。Y轴=亮度（0-100 IRE），X轴=画面位置。通过示波器可以精确判断：是否有高光削减（信号顶部出现水平线=信息丢失无法恢复）、暗部是否低于0（被截断为纯黑）、整体曝光是否在可接受范围内。
tags: [设备术语, 示波器, 曝光监控, 录像信号]
```

```yaml
id: g11
title: 矢量仪（Vectorscope）
type: glossary
source_chapter: 第9章 高清摄影
source_quote: "矢量仪要和示波器配合使用，测量录像信号的色度（色彩）。矢量仪的坐标是一个圆。"
summary: 矢量仪是色度（色彩）的测量工具，与示波器（亮度）配合使用。圆周方向=色相（角度），距圆心距离=饱和度。黑白图像显示为圆心一点。白平衡正确时，拍摄白色/灰色物体应在圆心。矢量仪是唯一可靠的色彩监控工具——显示器不可靠，但矢量仪数据永远值得信任。
tags: [设备术语, 矢量仪, 色彩监控, 白平衡]
```

```yaml
id: g12
title: 中灰滤镜（ND Filter - Neutral Density）
type: glossary
source_chapter: 第10章 曝光, 第13章 图像控制
source_quote: "这可以通过光圈、快门、进光量和中灰滤镜来控制，这些都是用来控制整个画面的曝光的。"
summary: 中灰滤镜均匀减少所有波长的光量，不改变色彩平衡。用途：在强光下使用大光圈（控制景深）、降低快门速度（创造运动模糊）、平衡室内外亮度差。分为固定密度和可变密度两种。中灰渐变镜（Graduated ND）只压暗画面的一部分（如天空），常用于外景。
tags: [滤镜术语, ND滤镜, 曝光控制, 光线管理]
```

```yaml
id: g13
title: 查找表（LUT - Look Up Table）
type: glossary
source_chapter: 第13章 图像控制
source_quote: "简单来说，查找表是一组预先计算好的数据，可以在录像设备色域和色度范围内，调整一张图像的色彩。"
summary: LUT是将RAW格式（或Log格式）图像转换为最终显示效果的预设数据表。1D LUT：每个色彩通道独立的伽马曲线。3D LUT：RGB三维晶格中的插值计算。用途：在片场预览最终效果、为后期调色提供参考、在不同显示设备间转换色彩空间。摄影指导经常创建自定义LUT。
tags: [后期术语, LUT, 色彩管理, RAW处理]
```

```yaml
id: g14
title: 分区曝光系统（Zone System）
type: glossary
source_chapter: 第10章 曝光
source_quote: "灰度上的每一区都和相邻的分区相差一档。第V区是中灰度，第0区是纯黑，第X区是纯白。"
summary: 分区曝光系统由安塞尔·亚当斯发明，将亮度分为0-X区（11级），每区相差一档。核心概念：定位（Place）=将某物体放到目标分区；跟随（Fall）=其他物体按亮度差自动落入对应分区。在电影摄影中，此系统用于精确控制画面中每个元素的亮度关系。
tags: [曝光术语, 分区系统, 亚当斯, 亮度控制]
```

```yaml
id: g15
title: 数码中间片（DI - Digital Intermediate）
type: glossary
source_chapter: 第13章 图像控制
source_quote: "数码中间片是利用数字技术和计算机能力的方法。拍摄时仍然使用胶片，然后转换到数字格式，利用数字领域所有的神奇技术进行图像处理。"
summary: DI工作流：胶片拍摄→高分辨率数字扫描（2K/4K）→数字调色和图像处理→转回胶片用于影院放映。优势：前所未有的图像控制力——可分区调色、逐帧调整、使用威力视窗、辅色矫正等。《逃狱三王》是第一部使用DI的电影。如果只以数字形式放映，则无需转回胶片。
tags: [后期术语, DI, 数码中间片, 色彩控制]
```
