# glossary.md — 关键术语提取

## G01: 交代镜头 (Establishing Shot)
```yaml
id: G01
title: 交代镜头
type: glossary
source_chapter: 第二十一章 词汇表
source_quote: "最常见的是在场景的开始以远景呈现，告诉观众场景的转换，和提示观众场景的气氛。"
summary: 在场景开始以远景呈现，告知观众场景已转换，提示气氛和物体的相关位置。是剪辑风格中最基础的空间建立工具。
tags: [交代镜头, 远景, 空间建立, 场景转换]
```

## G02: 主观镜头 (POV)
```yaml
id: G02
title: 主观镜头
type: glossary
source_chapter: 第十六章 视点、词汇表
source_quote: "第一人称叙述，可由希区柯克所使用的主观性技巧作为示范。"
summary: 透过角色眼睛来看事件的镜头。过量使用会在故事片中显得笨拙，因为只赋予一位角色的视觉观点。"视线剪切"是建立主观镜头的基本方法——先看，再看所看之物。存在完全主观和"修饰过的主观"两种程度。
tags: [主观镜头, 视线剪切, 希区柯克, 第一人称]
```

## G03: 切出镜头 (Cutaway)
```yaml
id: G03
title: 切出镜头
type: glossary
source_chapter: 第六章、词汇表
source_quote: "在主要动作或场景中间，偶尔插入一个单一的镜头。"
summary: 在主要动作中插入的单一镜头，打断主要动作的流动感。可介绍新的相关细节，或介绍与动作相关的新地点。也是处理越轴问题的过渡工具。
tags: [切出镜头, 过渡, 细节补充, 越轴]
```

## G04: 匹配剪辑 (Match Cut)
```yaml
id: G04
title: 匹配剪辑
type: glossary
source_chapter: 第二十一章 转场、词汇表
source_quote: "将两个相同动作的镜头连接在一起，制造连续动作的效果。"
summary: 将两个相同动作或形状相似的对象连接在一起的剪辑技巧。可以连接相同动作、相同对象或相似形状。戈达尔在《筋疲力尽》中将匹配剪辑与跳接结合——汽车和驾驶员维持原位但后景从白天转成黑夜。
tags: [匹配剪辑, 转场, 形状匹配, 戈达尔]
```

## G05: 动作轴线 (Axis of Action)
```yaml
id: G05
title: 动作轴线
type: glossary
source_chapter: 第六章
source_quote: "动作轴线原则下的180°工作区间。"
summary: 连接两个互相注视或互动的主体之间的假想线。摄影机应始终位于轴线的同一侧（180°工作区间内），以保持银幕方向的一致性。是剪辑风格中最基本的空间连续性规则。
tags: [动作轴线, 180度规则, 空间连续性, 剪辑风格]
```

## G06: 过肩镜头 (OTS)
```yaml
id: G06
title: 过肩镜头
type: glossary
source_chapter: 第六章、第九章、词汇表
source_quote: "以背向镜头人物的头和肩部作为画框，来框取面向镜头人物的取景方式。"
summary: 以前景人物的肩膀和头部作为画框来框取后景人物。是对话场景中最基本的构图方式，能统一空间（同时看到说话者和倾听者），维持轴线一致性。远摄镜头才能产生真正的"过肩"效果。
tags: [过肩镜头, 对话构图, 空间统一, 轴线]
```

## G07: 场面调度 (Mise-en-scène)
```yaml
id: G07
title: 场面调度
type: glossary
source_chapter: 第五章、词汇表
source_quote: "在拍摄时对场面和动作的控制，而非在后期剪辑阶段对空间的操控。"
summary: 法语"场景中的配置"。在拍摄时对场面和动作的控制，区别于后期剪辑的空间操控。包含演员走位、摄影机运动、景深安排等所有拍摄阶段的视觉决策。
tags: [场面调度, 拍摄阶段, 空间控制, 法语术语]
```

## G08: 景深摄影 (Deep Focus)
```yaml
id: G08
title: 景深摄影/深焦摄影
type: glossary
source_chapter: 第十三章、词汇表
source_quote: "以广角镜头和小光圈拍摄增加景深，使画面前后景皆保持清晰。"
summary: 使用广角镜头和小光圈使前后景都保持清晰的摄影风格。奥逊·威尔斯、威廉·惠勒和摄影师格列哥·托尔兰在《公民凯恩》中发展。常与纵深调度一起使用但不必然相关。可通过遮片合成、巨型道具等特殊技巧实现。
tags: [景深摄影, 深焦, 公民凯恩, 托尔兰]
```

## G09: 蒙太奇 (Montage)
```yaml
id: G09
title: 蒙太奇
type: glossary
source_chapter: 第二十一章 转场、词汇表
source_quote: "在美国，蒙太奇具有特别的意义，它代表一种浓缩的叙事工具。"
summary: 法语"组合"。在欧洲=所有剪辑；在苏联（爱森斯坦）=剪辑的精神；在美国=浓缩叙事工具（通常一分钟长，不用对白，以叠化短镜头讲述故事）。斯皮尔伯格在《大白鲨》中用6个3秒镜头压缩半小时。
tags: [蒙太奇, 叙事压缩, 爱森斯坦, 美国用法]
```

## G10: 镜流 (Shot Flow)
```yaml
id: G10
title: 镜流
type: glossary
source_chapter: 第八章
source_quote: "镜流，是指电影镜头的连接显现出流畅运动的效果。"
summary: 电影镜头连接所呈现的流畅运动效果。如河流般可以是汹涌的、宁静的、曲折的。景别和摄影角度是决定镜流的主要变化因素。"阅读故事板就是用眼睛进行剪辑"——训练在画面上移动眼睛的能力。
tags: [镜流, 流畅性, 剪辑节奏, 镜头连接]
```

## G11: 建构单位 (Building Block)
```yaml
id: G11
title: 建构单位
type: glossary
source_chapter: 第十二章
source_quote: "长镜头中大部分的动作，都是由这些重要的片刻串联起来的。"
summary: 运动镜头中的关键性片刻，代表了一个静态的调度形态。多个建构单位通过演员运动和摄影机运动连接成一个单一、不间断的镜头。每个单位对应一种形态（I/L/A）和位置。
tags: [建构单位, 运动镜头, 长镜头, 调度形态]
```

## G12: 反向运动 (Counter Move)
```yaml
id: G12
title: 反向运动
type: glossary
source_chapter: 第十二章
source_quote: "这种摄影机与人物运动方向相反的运动方式，被称为反向运动。"
summary: 摄影机与演员朝相反方向运动的调度技巧。是运动调度中最基本的组合方式——通过反向运动，可以在单一镜头内获得相当于多个固定镜头的视觉效果。
tags: [反向运动, 运动调度, 长镜头, 基本技巧]
```

## G13: 剪辑风格 (Continuity Style)

id: G13
title: 剪辑风格
type: glossary
source_chapter: 第一章、词汇表

## G14: 分像屈光度透镜 (Split-Field Lens)
```yaml
id: G14
title: 分像屈光度透镜
type: glossary
source_chapter: 第十三章
source_quote: "一个单一的屈光镜头，只是一个附着在主要光学镜头上的装置。"
summary: 附着在主镜头上的装置，只影响镜头一边的焦距，使镜头可以同时对焦于远处和近处的景物。在过肩镜头中特别有用——可以让前景人物后脑勺和后景人物都保持清晰。缺点是分割线处有可见模糊。
tags: [分像屈光度透镜, 焦距, 过肩镜头, 技术工具]
```

## G15: 仰拍/俯拍 (High/Low Angle)
```yaml
id: G15
title: 摄影角度
type: glossary
source_chapter: 第十四章
source_quote: "摄影角度本身是没有意义的。"
summary: 摄影机拍摄主体的角度选择。传统分析认为低角度=强势、高角度=弱势，但实际效果由故事上下文决定。在剧情片中有多种变化原因：跟拍人物、揭露/保留信息、改变视点、变化影像种类、建立定位、发展气氛。
tags: [摄影角度, 仰拍, 俯拍, 叙事功能]
```
