# Framework Extractor — 大师镜头 v1 (Master Shots Vol. 1)
# 阶段 1 产出：思维模型 / 决策框架 / 推理方法

- id: f01
  title: 长焦特效压缩空间框架
  type: framework
  source_chapter: 第1章 打斗镜头 §1.1
  source_quote: |
    "要拍好这一拳，秘诀就是：一定要用长焦镜头。长焦镜头有压缩空间的功效，使用长焦镜头，会让物体间的距离看起来比实际距离短。"
  summary: |
    面对需要制造"虚假暴力/追逐/接近"效果的场景时，使用长焦镜头压缩空间距离感。
    决策流程：(1) 判断场景是否需要"缩短距离"的错觉 → (2) 选择长焦镜头 → (3) 确保关键动作的落点藏在遮挡物后面 → (4) 演员配合时机。
    这是一个可迁移的"视觉欺骗"框架，适用于打斗、追逐、悬疑等多种场景类型。
  tags: [framework, lens-choice, spatial-compression, visual-deception]

- id: f02
  title: 编舞式镜头设计框架
  type: framework
  source_chapter: 前言
  source_quote: |
    "拍电影就像编舞，拍摄每一个场景你都要设计好演员和摄像师的舞步，当然，还要选对镜头。所有这些准备工作都做完了，你才能开始拍摄。"
  summary: |
    将每个场景的拍摄视为"双人编舞"：演员的运动路线和摄影机的运动路线必须事先协调设计。
    决策流程：(1) 设计演员的运动路线 → (2) 设计摄影机的配合路线 → (3) 确保两者在时间上同步 → (4) 排练至流畅。
    这是全书的元框架，贯穿所有100个技法——每个技法本质上都是"演员舞步+摄影机舞步"的一种特定组合。
  tags: [framework, choreography, meta-framework, camera-actor-sync]

- id: f03
  title: 镜头选择决策框架（焦距→效果映射）
  type: framework
  source_chapter: 贯穿全书（每章均强调）
  source_quote: |
    "书中的每一章我都会强调镜头的选择，并告诉你选用哪种镜头好。就算你对镜头不感兴趣，我还是建议你掌握一些基本的知识。"
  summary: |
    面对任何拍摄场景时，首先根据需要的视觉效果选择镜头焦距：
    - 长焦：压缩空间、虚化背景、制造亲密/紧张感、增强特写情感力度
    - 广角/短焦：夸大动作幅度、加大景深、制造速度感和空间感
    - 中焦：保持自然、适合对话和日常场景
    这个框架在全书100个技法中反复出现，是所有镜头设计的基础决策树。
  tags: [framework, lens-choice, decision-tree, foundational]

- id: f04
  title: 分散注意力-打破预期框架
  type: framework
  source_chapter: 第7章 惊惧镜头 §7.2
  source_quote: |
    "基本的原则是：先制造一点紧张的气氛，然后让观众放松一下，当他们刚松一口气时，马上出招吓坏他们。要想让观众放松，可以用镜头之外的事物吸引他们的注意力。"
  summary: |
    制造惊吓/悬疑效果的通用推理方法：
    (1) 制造紧张气氛 → (2) 用镜头外的事物分散观众注意力 → (3) 观众注意力转移时放松警惕 → (4) 在放松瞬间出其不意地打破预期。
    这个"紧张-放松-突袭"的三段式结构适用于所有需要"出其不意"效果的场景，不仅限于恐怖片。
  tags: [framework, suspense, audience-psychology, attention-control]

- id: f05
  title: 连贯镜头优于剪辑框架
  type: framework
  source_chapter: 第6章 现身和发现 §6.1
  source_quote: |
    "好导演一般都尽可能保持镜头连贯又干脆，不会依赖太多的剪辑。连贯的镜头会让观众有想看的欲望，视觉效果也更强烈。"
  summary: |
    面对"如何在一个镜头中表现多个空间/事件"的问题时，优先考虑用一个连贯镜头完成，而非依赖后期剪辑切换。
    决策流程：(1) 能否用一个镜头交代清楚？→ (2) 能否利用反射/门框/前景遮挡同时表现两个空间？→ (3) 只有在连贯镜头无法完成时才使用剪辑。
    这个框架强调：连贯镜头叙事更清楚、视觉效果更强烈、观众体验更沉浸。
  tags: [framework, continuity, single-take, anti-editing-dependence]

- id: f06
  title: 视角高度→情感认同框架
  type: framework
  source_chapter: 第5章 情节变化 §5.2 / 第4章 §4.5
  source_quote: |
    "摄影机的位置要比演员的头低一点，向上仰拍他的脸，使用短焦镜头，这样，演员只要后退几步，就能一下子从特写变小。"
  summary: |
    摄影机的高度直接影响观众对角色的情感认同：
    - 仰拍（低于角色）→ 赋予力量感、威严感
    - 俯拍（高于角色）→ 表现弱势、被压迫感
    - 平视（与角色同高）→ 建立平等、亲密关系
    - 极低角度 → 空间显得局促、幽暗，增强紧张感
    这个框架在全书中反复出现，是每个场景设计时摄影机定位的首要考量。
  tags: [framework, camera-height, emotional-identification, power-dynamics]

- id: f07
  title: 双机位对比张力框架
  type: framework
  source_chapter: 第2章 追逐镜头 §2.4 / §2.9
  source_quote: |
    "用两台摄影机用不同的速度追着演员拍。这样拍出来的画面会有一种错觉，让人觉得主角快要被抓住了，可实际上他却没被追上。"
  summary: |
    通过两台摄影机的不同运动状态制造对比张力：
    - 一台手持（慌乱感）+ 一台推车（稳定感）→ 稳定的一方占上风
    - 一台快速跟拍 + 一台慢速后退 → 制造"快要被追上"的错觉
    - 不同焦距搭配 → 一台压缩空间，一台夸大速度
    这个框架的核心是：对比产生张力，运动状态的差异暗示力量对比。
  tags: [framework, dual-camera, tension, contrast-technique]

- id: f08
  title: 演员静止+背景运动=重大决定框架
  type: framework
  source_chapter: 第5章 情节变化 §5.3
  source_quote: |
    "演员仿佛是静止不动的，而她身后的背景却在飞速地旋转。这种旋转是幻觉……这样拍就是想让观众感到眩晕。"
  summary: |
    当需要表现角色做出重大决定/转折时，使用"演员静止+背景运动"的对比：
    演员走半圆弧线，摄影机从其运动轨迹中穿过并拉远，长焦镜头+快速移动产生背景飞速旋转的幻觉。
    适用场景：打斗前的亮相、情节重大转折、角色决心的表现。
    关键限制：只在真正戏剧性的时刻使用，频繁使用会失去效果。
  tags: [framework, decisive-moment, background-motion, dramatic-turning-point]

- id: f09
  title: 主观视角→客观视角切换框架
  type: framework
  source_chapter: 第4章 悬疑搜索 §4.7
  source_quote: |
    "要拍好搜寻的情景，更好的方法是跟着演员拍，让观众产生身临其境的感觉。两次拍摄，一次从演员身后拍，一次拍他的视角。"
  summary: |
    表现搜索/发现/探索场景时，在主观视角（演员眼睛位置）和客观视角（演员身后跟随）之间快速切换：
    (1) 从演员身后跟拍 → 建立"跟随"关系
    (2) 切到演员的主观视角 → 让观众"看到"演员所见
    (3) 快速在两者间切换 → 制造紧迫感和身临其境感
    (4) 最终切回演员反应镜头 → 完成情感闭环
  tags: [framework, POV-switching, search-scene, immersion]

- id: f11
  title: 障碍物渐现登场框架
  type: framework
  source_chapter: 第3章 角色登场退场 §3.4
  source_quote: |
    "让他静静地出现在屏幕中央……让主角站在屏幕中间，前面先用东西挡着，然后障碍物渐渐移开，主角就静静地出现了。"
  summary: |
    重要角色登场的通用设计框架：
    (1) 主角定位于画面中央 → (2) 用障碍物（门、车、人群）遮挡 → (3) 障碍物缓慢移开 → (4) 摄影机保持静止 → (5) 主角现身后的定格静止。
    门是最常用的道具：双开门比单开门更有力量，推拉门可以逐渐展示。
    这个框架的核心是"延迟满足"——越是重要的角色，越需要缓慢地揭示。
  tags: [framework, character-entrance, reveal, delayed-gratification]

- id: f12
  title: 长焦+前景虚化=注意力聚焦框架
  type: framework
  source_chapter: 第8章 引导注意力 §8.6 / 第6章 §6.3
  source_quote: |
    "使用长焦镜头，因为长焦镜头对焦的范围小，这样在拍摄时，只有主角是对上焦的，其他人都是虚化的。"
  summary: |
    需要观众注意力集中到特定对象时的通用方法：
    (1) 使用长焦镜头缩小对焦范围 → (2) 确保目标清晰、周围虚化 → (3) 给目标更强的光照 → (4) 让目标与背景形成色彩/亮度对比。
    人眼本能地被"明亮+清晰"的物体吸引，利用这个特性可以在不移动摄影机的情况下引导观众注意力。
  tags: [framework, attention-guiding, long-lens, selective-focus]
