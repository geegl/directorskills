- id: f01
  title: 场景转换四功能诊断法
  type: framework
  source_chapter: 第 1 章
  source_quote: |
    场景转换的四种功能：1.让观众思考 2.让观众感受 3.让观众舒缓或形成张力 4.让观众与人物产生联系
  summary: |
    面对任何一个场景转换决策时，先诊断它应满足四种功能中的哪一种（或几种）：
    思考、感受、舒缓/形成张力、与人物联系。
    不同功能导向完全不同的转换手法选择。
    这是从"本能"到"技艺"的第一步判断框架。
  tags: [framework, diagnostic, decision, scene-transition]

- id: f02
  title: 投入公式（投入=情节张力+情绪化）
  type: framework
  source_chapter: 第 4 章
  source_quote: |
    投入=情节张力+情绪化。当观众接受全新情节信息时各种不同情绪化的内心张力。
  summary: |
    衡量观众在场景转换中投入程度的操作性公式。
    投入上升发生在场景A结尾（情感瞬间），沉思发生在场景B开头（放松）。
    两个变量（情节张力和情绪化）的组合决定了观众的参与深度。
    用于诊断场景转换的情感效果是否到位。
  tags: [framework, formula, engagement, emotion]

- id: f03
  title: 沉思公式（沉思=故事背景+情绪）
  type: framework
  source_chapter: 第 4-5 章
  source_quote: |
    沉思=缺乏情节张力的客观事件。故事背景与情绪的结合。
  summary: |
    场景低潮中观众内省程度的公式。
    当情节张力下降时，通过注入故事背景和情绪来维持观众的内在参与。
    与投入公式互补：投入是向上的力量，沉思是向下的力量，
    两者交替形成场景转换的"波浪"节奏。
  tags: [framework, formula, contemplation, rhythm]

- id: f04
  title: 二元对立场景并置工具
  type: framework
  source_chapter: 第 2 章
  source_quote: |
    索绪尔认为大脑通过指出概念不是什么加以比较就使得概念产生意义。叙事者为了阐释清楚，最好是运用对立。
  summary: |
    将索绪尔语言学的二元对立操作化为场景设计工具。
    当两个截然不同的场所被并置时，"摩擦"自动产生戏剧效果。
    对立清单包括：黑夜/白天、平静/嘈杂、忙碌/安逸、热闹/荒僻、
    室内/户外、近/远、恐惧/有趣。
    对立可以划分场景边界、激发意义、强调之前的事件。
  tags: [framework, binary-opposition, scene-design, saussure]

- id: f05
  title: 泡泡理论（场景缺失激发想象）
  type: framework
  source_chapter: 第 2 章
  source_quote: |
    当一个场景缺失时，就会让我们思索这曾是怎样的一个场景。
  summary: |
    通过刻意省略预期中的场景来强调该场景的重要性。
    当场景缺失时，观众会主动回想和想象，从而加深对叙事的参与。
    切至黑屏是最极致的形式——停止时间，创造内容与故事的缺失。
    白屏则相反，是故事的强烈表现，完全表现当下的瞬间。
  tags: [framework, bubble-theory, omission, audience-imagination]

- id: f06
  title: 格式塔省略原理
  type: framework
  source_chapter: 第 8 章
  source_quote: |
    格式塔理论：缺失的信息激发想象力填充，大脑内化创建的信息。
  summary: |
    将格式塔心理学应用于电影叙事中的"省略"技法。
    故事以外的内容与保留的内容同样重要——省略的场景比展示的场景
    更能让观众主动参与意义构建。
    硬切、叠化、划变、黑场淡出都是不同程度的格式塔省略工具。
    库布里克《2001太空漫游》跨越几十年的场景是最经典的案例。
  tags: [framework, gestalt, omission, time-manipulation]

- id: f07
  title: 亲近轴与频率轴双维度调控
  type: framework
  source_chapter: 第 5 章
  source_quote: |
    亲近轴=特写到广角的范围。频率轴=快速剪辑到长镜头的范围。保持近景+快速剪辑=维系张力。广角+长镜头=缓解张力。
  summary: |
    两个正交维度控制场景转换的张力水平：
    亲近轴（特写↔广角）和频率轴（快速剪辑↔长镜头）。
    近景+快速剪辑=维系张力；广角+长镜头=缓解张力。
    这是一个可操作的二维决策矩阵，用于设计场景转换的节奏。
  tags: [framework, proximity-axis, frequency-axis, tension-control]

- id: f08
  title: 转化策略四象限（音乐/静默组合）
  type: framework
  source_chapter: 第 6 章
  source_quote: |
    转化策略的变体：1.静默到音乐 2.音乐到音乐 3.使用预录歌曲 4.歌词的作用
  summary: |
    场景转换中最重要的音频决策框架。
    四种音乐/静默组合各有不同的情感效果：
    静默→音乐=压抑情感释放，宣泄形式；
    音乐→音乐=情感变化，保持和强化能量；
    预录歌曲=与观众直接沟通；
    歌词=类似叙述者，代表人物说话或评论。
  tags: [framework, music, audio-strategy, scene-transition]

- id: f09
  title: 关联理论（相似性关联）
  type: framework
  source_chapter: 第 3 章
  source_quote: |
    苏联导演普多夫金提出关联理论：如原子以化学连接的方式相连，电影镜头以相似的方式关联。
  summary: |
    大脑习惯于寻找模式和联系，利用相似性将两个场景关联起来。
    四种关联维度：关联物体、关联人物、关联场所、关联镜头。
    库布里克的骨头→太空站是最著名的案例。
    关联是让观众"不会从火车上掉下"的基本方式。
  tags: [framework, linkage, similarity, montage]

- id: f10
  title: 场景转换力度决策框架
  type: framework
  source_chapter: 第 1 章
  source_quote: |
    最简单的场景转换就是一个硬切。整段的新场景和段落可以用来桥接两个主要场景。
  summary: |
    判断每个场景转换需要多大强调力度的决策谱系：
    一端是最简单的硬切，另一端是完整的场景转换段落（如《角斗士》8分钟）。
    需要根据场景在整体叙事中的位置（六种主要转换）来决定力度。
    不是每个场景转换都需要与众不同。
  tags: [framework, decision, intensity, scene-transition]

- id: f11
  title: 六种主要场景转换分类
  type: framework
  source_chapter: 第 2 章
  source_quote: |
    六种主要的场景转换：开场、第一幕结尾、第二幕中段、第二幕结尾、高潮之后、离场。
  summary: |
    按照叙事结构中的位置，将场景转换分为六种类型：
    1.开场（现实→戏剧世界）2.第一幕结尾（旅程开始）
    3.第二幕中段（方向转换）4.第二幕结尾（最低潮）
    5.高潮之后（张力释放）6.离场（故事→现实）。
    每种类型需要不同的处理策略和力度。
  tags: [framework, classification, narrative-structure, six-types]

- id: f12
  title: 嘻哈蒙太奇与时间拼贴
  type: framework
  source_chapter: 第 8 章
  source_quote: |
    时间拼贴：缩短的迷你场景关联两个场景。颠倒时间：嘻哈蒙太奇硬切展示日常。
  summary: |
    两种高级时间操纵框架：
    时间拼贴=在一个小空间中组合大量时间片段，压缩时间；
    嘻哈蒙太奇=快速硬切展示主角日常例行工作，产生狂乱感觉。
    两者都是通过碎片化处理来加速叙事节奏的技法。
  tags: [framework, temporal-collage, hip-hop-montage, time-compression]

- id: f13
  title: 三角色视角框架（编剧/导演/剪辑师）
  type: framework
  source_chapter: 全书
  source_quote: |
    导演将场景转换放到电影中，编剧在剧本里放置场景转换来进行铺垫，剪辑师使用场景转换激发灵感。
  summary: |
    每一种场景转换技法都可以从三个角色视角分别实施：
    编剧在剧本中预设转换，导演在拍摄中决定保留或删减，
    剪辑师在后期中创造流畅连接。
    这是一个贯穿全书的元框架，用于将任何技法转化为可执行的操作。
  tags: [framework, three-roles, writer-director-editor, meta-framework]

  title: 场景DNA（长度×张力×情绪）
  type: framework
  source_chapter: 第 7 章
  source_quote: |
    结合长度、张力、情绪创造电影DNA。
  summary: |
    将每个场景拆解为三个维度——长度、张力、情绪，
    然后将这些维度的组合视为电影的"DNA"。
    通过调整这三个维度的排列顺序，创造独特的节奏模式。
    场景长度变化趋势：1960年前平均3分钟，之后越来越短。
